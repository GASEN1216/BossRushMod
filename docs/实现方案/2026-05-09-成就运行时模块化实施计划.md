# 成就 Runtime Module Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在不改变游戏内行为的前提下，把方向文档阶段 2 中的成就 UI 快捷键和成就视图初始化收束到成就模块边界。

**Architecture:** 成就系统已有实现保留在 `Achievement/` 目录和现有 partial 方法中。新增 `AchievementRuntimeModule` 只负责每帧热键 tick；初始化和清理通过 `AchievementRuntimeHooks` wrapper 在 `Awake` / `OnDestroy` 原位置调用，避免改变事件订阅和 UI 初始化时序。

**Tech Stack:** C# 7.3、Unity Mono、Duckov ModBehaviour、手工 `compile_official.bat` 编译清单、Python 静态 guard。

---

## Scope

本计划覆盖：

- 成就系统初始化 wrapper。
- 成就视图 `AchievementView.EnsureInstance()` wrapper。
- 成就 UI 快捷键 tick wrapper。
- 成就事件清理 wrapper。
- runtime module 只转发成就 UI 快捷键 tick。

不覆盖：

- 不改成就定义、解锁条件、Steam 弹窗、成就勋章物品。
- 不改 DevMode F4 清空成就调试入口；该入口属于 DebugTools wrapper。
- 不改变 `Health.OnHurt += OnPlayerHurtForAchievement` 的订阅时机，也不把它移入 runtime module。

## Task 2: Runtime Module And Wrappers

- [ ] **Step 1: Create module**

```csharp
namespace BossRush
{
    internal sealed class AchievementRuntimeModule : BossRushRuntimeModuleBase
    {
        private ModBehaviour owner;

        public override string ModuleName
        {
            get { return "Achievement"; }
        }

        public override void OnAwake(ModBehaviour owner)
        {
            this.owner = owner;
        }

        public override void OnUpdate(float deltaTime, float unscaledDeltaTime)
        {
            if (owner == null)
            {
                return;
            }

            owner.TickAchievementRuntime(deltaTime, unscaledDeltaTime);
        }

        public override void OnDestroy()
        {
            owner = null;
        }
    }
}
```

- [ ] **Step 2: Create wrapper file**

```csharp
using System;
using Duckov.UI;
using UnityEngine;

namespace BossRush
{
    public partial class ModBehaviour
    {
        internal void InitializeAchievementRuntime()
        {
            InitializeAchievementSystem();
            AchievementView.EnsureInstance();
        }

        internal void TickAchievementRuntime(float deltaTime, float unscaledDeltaTime)
        {
            try
            {
                UnityEngine.KeyCode achievementKey = UnityEngine.KeyCode.L;
                if (config != null && config.achievementHotkey > 0)
                {
                    achievementKey = (UnityEngine.KeyCode)config.achievementHotkey;
                }

                if (UnityEngine.Input.GetKeyDown(achievementKey))
                {
                    if (Duckov.UI.View.ActiveView == null)
                    {
                        AchievementView.Instance.Toggle();
                    }
                }
            }
            catch (Exception e)
            {
                DevLog("[BossRush] 成就界面快捷键处理失败: " + e.Message);
            }
        }

        internal void CleanupAchievementRuntime()
        {
            Health.OnHurt -= OnPlayerHurtForAchievement;
            UnsubscribeAchievementEvents();
        }
    }
}
```
