## v2.1.36

### 发布日期
- 2026-04-25

### 主题
- **阿稳扫箱交互、通关奖励箱高光缓降、Boss 筛选覆盖 Mode D/E/F、原版概率掉落作用域修复**：本次更新为阿稳新增常驻 `扫箱` 交互，用于快速收纳 / 整理 / 清除 BossRush 战利品箱；标准 BossRush 通关奖励改为高光箱子在玩家头顶显现后缓慢降落；Boss 筛选器现在也会同步过滤白手起家 / 划地为营 / 血猎追击；`useLegacyBossLootProbabilities` 的作用范围修正为不再只限标准 BossRush。

### 详细更新日志

#### 新增
- 阿稳新增交互 **`扫箱`**：可针对当前场景里的 BossRush 战利品箱进行集中整理，并统一生成一个阿稳代收箱，便于快速收纳 / 整理 / 清场。

#### 优化
- 标准 BossRush 通关后的奖励箱不再直接在路牌旁发放，而是先以高光虚影的形式出现在玩家头顶，随后缓慢凝实并落地。
- Boss 筛选器的禁用结果现在也会同步作用于 **白手起家 / 划地为营 / 血猎追击** 的 Boss 池。

#### 修复
- 修复配置项 `useLegacyBossLootProbabilities` 之前只作用于标准 BossRush 的问题；现在会按设计同步影响白手起家 / 划地为营 / 血猎追击中的对应常规掉落品质分布。
