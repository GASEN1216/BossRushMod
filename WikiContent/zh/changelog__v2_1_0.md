## v2.1.0

### 发布日期
- 2026-02-10

### 同期重点
- 这是一个真正意义上的**大版本**。
- 龙王、龙王套装、飞行图腾、逆鳞、成就系统、重铸系统、哥布林好感度与礼物系统，在这一时期集中成形。
- 也是 BossRush 从“挑战玩法扩展”转向“战斗 + 养成 + NPC + 长期目标”综合型 Mod 的关键节点。

### 本版主要方向
- **龙王线**：龙王 Boss、龙王套装、专属掉落、音效、三阶段与太阳舞、飞行图腾、龙王冲刺。
- **重铸线**：冷淬液、固定属性、保底、持久化、UI、支持 Stat 与 Variable。
- **成就线**：成就系统、成就勋章、页面优化、资源桶与弹窗。
- **NPC 关系线**：哥布林好感度、礼物 UI、叮当涂鸦、钻石 / 砖石、好感度衰减、长对话持久化。
- **底层能力**：序列化、SavesSystem 接入、图腾位、物品工厂、掉落黑名单与本地化整理。
