## 龙王套装

### 这是啥？

龙裔套装的**终极进化形态**——焚天龙皇掉落。保留火焰转治疗核心被动，冲刺能力直接拉满，还移除了龙裔套装的毒素易伤和视野缩小。**拿到就是毕业。**

### 套装部件

#### 龙王之冕（头盔）
- 护甲：7 / 耐久：200 / 物理抗性：-15% / 暴击伤害：+15%
- 风暴抗性：+1 / 寒冷抗性：+1 / 火焰抗性：-20% / 电击抗性：-20%
- 掉落：焚天龙皇（**15% 掉率**）
- 相比赤龙首：暴击更高，没有毒素易伤，没有视野缩小

#### 龙王鳞铠（护甲）
- 护甲：7 / 耐久：200 / 物理抗性：-25%
- 风暴抗性：+1 / 寒冷抗性：+1 / 火焰抗性：-25% / 电击抗性：-25%
- 掉落：焚天龙皇（**15% 掉率**）
- 相比焰鳞甲：没有毒素易伤

### 两件齐穿——龙王模式启动

- **火焰转治疗** — 火焰伤害不扣血，反而回复 80% 等量生命值
- **龙眼光效** — 头部出现红色呼吸灯，帅就完事了
- **龙王冲刺** — 6 米冲刺，冷却仅 0.5 秒
- **二段冲刺** — 首段冲刺后 0.15 秒内按方向键可追加 3 米
- **熔浆轨迹** — 冲刺路径留下熔浆灼烧（每 0.5 秒 5 点伤害，持续 3 秒），**一边跑一边烧敌人**
- **金橙残影** — 冲刺时留下金橙色残影（比龙裔套装的橙红色更霸气）

### 比龙裔套装强多少？

| 属性 | 龙裔 | 龙王 |
|------|------|------|
| 冲刺距离 | 3 米 | 6 米 + 3 米二段 |
| 冲刺冷却 | 1.5 秒 | 0.5 秒 |
| 熔浆轨迹 | 无 | 有 |
| 毒素易伤 | 有 | 无 |
| 视野缩小 | 有 | 无 |
| 暴击伤害 | +10% | +15% |
