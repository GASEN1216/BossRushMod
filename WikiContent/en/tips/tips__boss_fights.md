## Boss Fight Guide

### Dragon Descendant — Medium

#### Phase 1
- Damage only 0.3x — low pressure. Learn the patterns
- Every 10th bullet = rocket (1m blast). Keep distance
- Incendiary bomb every 5s — dodge the ground fire
- Steady DPS, no rush

#### Phase 2 (50% HP "Death" → Revival)
- **Don't celebrate.** It comes back harder: 1.1x dmg, 1.5x crit, 1.5x chase speed
- Contact damage now active — don't let it touch you
- Firebombs every 1s, throws at itself to heal
- **Key: out-DPS the self-healing**

#### Counters
- Dragon Set / Dragon King Set = fire heals you. Fight becomes trivial
- Ice weapons slow it in Phase 2
- Reverse Scale as safety net

#### No-Hit Tips
- Phase 1: mid-range, dodge rockets + firebombs
- Phase 2: dash to create distance, never let it close
- Dragon Set = immune to fire, just dodge bullets + contact
- Boss Filter → solo it

### Skyburner Dragon Lord — Hardest

#### General
- It flies (3-5m) — **use guns, not swords**
- It follows you — don't try to outrun it
- **Never stop moving**

#### Skill Responses
- **Prismatic Bolt** — Move laterally for 2.5s to shake tracking
- **Charge** — See ground ring → dodge perpendicular immediately
- **Solar Dance** — Find gaps between rotating beams, or run out of range
- **Rainbow Ring** — Run OUT during expansion, IN during contraction
- **Phantom Spear** — Read warning lines, stand between them

#### Phase 2
- Attack interval halved (1.0s → 0.5s)
- Charge becomes double charge
- New Spiral Bolt + Spear Barrage (4 × 16 spears)
- Requires faster reactions

#### "Protect me, my child!"
- At 1 HP → ascends, summons half-stat Dragon Descendant
- Fires Bolts every 3s while Junior lives
- **Kill the Descendant first**

#### Recommended Gear
- Dragon Set / Dragon King Set (fire immunity + dash)
- High DPS guns (shorter fight = fewer mistakes)
- Cloud Rider (aerial evasion)
- Reverse Scale (emergency save)

#### No-Hit Tips
- Memorize all telegraphs and safe zones
- Dragon King dash (0.5s CD, 6m) is the best evasion tool
- Phase 2 = double tempo, biggest challenge
- "Protect me" phase = manage two targets simultaneously
- Boss Filter → solo practice

### General Tips

- Use arena cover and terrain
- Ammo shop near signpost — restock anytime
- Multi-boss waves: watch directional indicators, don't get surrounded
- Boss Filter to disable problematic Bosses
- Kill speed affects drop quality

[tip] Dragon Set / Dragon King Set = fire heals you. Huge advantage against both dragon Bosses.
