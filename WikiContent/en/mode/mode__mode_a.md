## Mode A: Easy

### Core rules
- One boss per wave.
- Finite and straightforward structure.
- Best mode for first clears, first boss reading, and first drop farming.

### Why it matters
- This is where you learn signpost flow, wave rhythm, lootbox handling, and boss recognition.
- It is not just "baby mode". It is the cleanest training environment.

### Best use cases
- First-time players.
- Boss practice with a smaller boss pool.
- Controlled farming without too much battlefield noise.

[tip] If you want to learn a specific boss, shrink the boss pool first, then use this mode.
