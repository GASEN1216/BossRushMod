## Mode B: Standard

### Core rules
- Three bosses of the same type per wave.
- Finite structure, but much more pressure than Easy.
- This is where the mod starts asking for real target priority and movement discipline.

### What changes compared with Easy
- Mistakes snowball faster.
- Positioning matters more than raw damage.
- Bad terrain choices punish you harder.

### Best use cases
- Players who already understand the basic BossRush loop.
- Faster farming once you are stable.
- Pressure testing your current build before entering harder modes.

[tip] If Easy taught you what bosses do, Standard teaches you how to keep control when several problems arrive at once.
