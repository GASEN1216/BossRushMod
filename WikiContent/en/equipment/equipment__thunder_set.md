## Thunder Set

[warn] Developer Preview: This set exists in the game database but has **no obtain path in the current version** (no drop, no shop, no reward). Only obtainable via dev/debug grant; an acquisition route is planned for a future update.

### What Is It?

A **2-piece lightning set** composed of the **Thunder Horn** helmet and **Thunder Armor** body piece. Each item alone gives modest armor, but equipping both activates **Thunder's Wrath** — electricity resistance plus a close-range AOE lightning counter.

| **Thunder Horn** | Helmet | 6 | Head Armor +5 | 500055 |
| **Thunder Armor** | Body Armor | 6 | Body Armor +5 | 500056 |

- Durability: 999 (effectively permanent)
- **Drop**: No standard obtain path in current version
- Won't drop on death

### 2-Piece Effect: Thunder's Wrath

Activates when both pieces are equipped simultaneously (screen notification appears).

**Passive:**
- Electricity damage resistance **+50%**

**Reactive (counter AOE):**
- When struck by an attacker within **6 meters**, **25%** chance to release a **lightning AOE** at your location
  - **4m radius** / **25 electricity damage**
- Maximum once every **3 seconds**
- Does **not** damage yourself

### Frost Set vs. Thunder Set

| | Frost Set (Frost Ward) | Thunder Set (Thunder's Wrath) |
|--|--|--|
| Resistance | Ice | Electricity |
| Trigger range | 5m | 6m |
| Trigger chance | 30% | 25% |
| Effect | Freeze attacker | AOE damage blast |
| Cooldown | 5s | 3s |

### Tips

- Electricity resistance helps against Harasser enemies in Zombie Mode
- 3s cooldown fires more frequently than Frost Set — better in sustained multi-target fights
- 4m radius AOE can hit multiple clustered enemies at once — great in Infinite Hell or Zombie Mode waves
- Counter damage (25) is modest per hit but adds up over a long fight

[tip] Thunder Set shines in dense multi-enemy waves (Infinite Hell, Zombie Mode normal waves). Frost Set is better for 1v1 boss control. Pick based on the run type.
