# 详细更新日志

# 新增：星愿许愿台弹幕

- 打开「布满了灰尘的星愿许愿台」面板时，会在遮罩之上、面板之下显示滚动弹幕层。
- 弹幕内容来自已保存的心愿记录，优先使用最新拉取结果；若联网暂时失败，则会回退到本地缓存。
- 弹幕层使用对象池与分泳道滚动，尽量减少首次打开时的卡顿和运行中抖动。

# 修复：物品无法加载的问题

- 修复部分 BossRush 自定义装备与道具在 Mod 启用/禁用后暂时恢复正常、但完整重启游戏后又重新变成白底问号占位图标的问题。
- 现在存档还原、仓库、商店与 UI 在按 TypeID 查询这些动态物品时，会先走按需注册兜底，降低 prefab 尚未注册时回退成占位物品的概率。
