# v2.1.27

# 发布日期
- 2026-03-23

# 本版主要方向
- **Boss 池与重铸清理**：Boss 筛选器不再混入 Mode E / Mode F 运行时小怪，重铸池也会排除不该被洗的内部时序属性并自动清理旧异常数据。
- **快递、婚姻与交互体验补强**：一键存入/寄存跳过锁定格，配偶自动回家提示更明确，多选 NPC 交互更稳定。
- **Mode E / F 显示与商店同步**：划地为营与血猎追击的名称显示、奖励说明和商店分类继续统一，面具分类现包含耳机。

# 详细更新记录
# 2026-03-21
- Boss 筛选器会自动剔除 Mode E / Mode F 运行时生成的小怪污染，并保留龙裔 / 龙王相关特殊 Boss 预设。
- 重铸系统不再将近战出伤时序属性纳入重铸池；旧版本遗留的异常重铸数据会在读取时自动清理并回滚到预制体默认值。
- 修复 NPC 单点多选交互组初始化，哥布林、护士、阿稳与婚礼教堂的多选交互更稳定。

# 2026-03-23
- 一键存入 / 一键寄存排除锁定格子。
- 配偶因好感跌出 Lv.10 而自动回家时，会额外显示玩家气泡提示。
- 划地为营与血猎追击神秘商人的面具分类现在同时售卖耳机。
- 血猎追击的悬赏额外掉落与撤离奖励统一按共享高品质奖励池规则结算。
- 继续整理 Mode E / F 的血条名称、悬赏显示与工事系统，并同步相关 Wiki 说明。
