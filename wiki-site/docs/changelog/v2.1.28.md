# v2.1.28

# 发布日期
- 2026-03-27

# 本版主要方向
- **死亡亡魂系统上线**：玩家死亡后会在原地留下一个继承外观、装备与战斗风格的亡魂记录；再次回到同一地图、同一子场景时，就会迎来一次“和自己对决”的补课战。

# 详细更新记录
- 新增死亡亡魂系统。
- 玩家在正常游戏场景死亡后，会记录死亡位置、外观、装备、声音与武器状态。
- 再次进入同一地图、同一子场景时，亡魂会在原死亡位置刷新。
- 亡魂会根据你死亡时携带物品价值占总财产的比例，分为强壮、均衡、弱小三档。
- 三档亡魂拥有不同的生命、伤害和移动强度；当前弱小亡魂也会有轻微速度加成。
- 亡魂会根据持枪或近战状态，自动切换枪战型或狼系近战型行为模板。
- 亡魂现在不会在玩家一进图时立刻追杀，而是会像原版敌人一样自然发现目标后再开打。
- 亡魂会显示独立名字和血条，击败后本次记录会被清除。
- 亡魂本身不会掉落战利品箱。
