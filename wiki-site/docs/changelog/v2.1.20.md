# v2.1.20

# 发布日期
- 2026-03-14

# 同期重点
- 这是继 `v2.1.0` 之后的第二个超大版本阶段。
- **护士 NPC、治疗、婚礼与关系系统后续、焚皇断界戟、龙皇焚天龙铳、Mode E 强化、37号实验区与迷宫、敌怪异常救援监控、重铸系统优化** 都集中在这一时期完成。
- 这也是当前代码基线最接近的更新汇总页。

# 本版主要方向
- **NPC 与关系线**：护士 NPC、治疗、安神滴剂、平安护身符、婚礼教堂、钻石戒指、婚后链路和剧情键修复。
- **龙王武器线**：焚皇断界戟持续优化，龙皇焚天龙铳从初版推进到数据驱动 Profile 架构。
- **Mode E**：新增猎王响哨与血狩烽火，商人子弹商店修复，刷自定义 NPC，强制唤醒 Boss，部分阵营参数改用白手起家参数。
- **地图与场景**：新增 37号实验区与迷宫。
- **稳定性与底层**：敌怪卡地 / 掉图恢复监控、教堂卡顿优化、标签与本地化修复、重铸系统继续打磨。
