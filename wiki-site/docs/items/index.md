# 物品总览

BossRush Mod 添加了 20 余种自定义物品，按用途分为以下几类。

## 物品分类

| 分类 | 说明 | 详情页 |
|------|------|--------|
| [入场与功能物品](key-items.md) | 进入各模式的凭证、百科书、成就勋章等 | → |
| [NPC 相关物品](npc-items.md) | 与 NPC 互动相关的礼物、材料、特殊道具 | → |
| [消耗品](consumables.md) | 治疗、增益、召唤类消耗品 | → |
| [模式专属物品](mode-items.md) | 划地为营和血猎追击的专属道具 | → |

装备类物品（套装、图腾、武器）请参见 [装备总览](../equipment/index.md)。

## 物品获取途径

| 途径 | 说明 |
|------|------|
| 基地商人 | 船票、营旗、血猎收发器、冒险家日志、成就勋章、砖石等 |
| NPC 叮当 | 钻石、钻石戒指（好感度达标后解锁）、冷淬液 |
| NPC 羽织 | 安神滴剂（好感度奖励）、平安护身符（好感度奖励） |
| Boss 掉落 | 龙裔遗族和焚天龙皇的专属装备掉落 |
| 战利品箱 | 击杀 Boss 后随机掉落的各类物品 |
| 成就奖励 | 现金奖励（通过成就面板领取） |
| 系统赠送 | 叮当涂鸦（好感度满级礼物）、模式内自动发放的消耗品 |
| 神秘商人 | 划地为营和血猎追击模式内的分类商店 |
