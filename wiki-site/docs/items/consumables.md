# 消耗品

## 生日蛋糕

特殊活动物品，可以作为礼物送给 NPC。

| 属性 | 说明 |
|------|------|
| 获取方式 | 特殊活动 |
| 好感度加成 | 送给任意 NPC 时 +150 |
| 类型 | 消耗品 |

生日蛋糕是好感度收益最高的常规礼物之一（仅次于钻石戒指的 +500），如果你在活动期间获得了它，建议送给正在培养好感度的 NPC。

## 安神滴剂

羽织亲手调配的安神滴剂，使用后清除大部分负面 Buff。

| 属性 | 说明 |
|------|------|
| 获取方式 | 羽织好感度 Lv.3 一次性奖励 5 瓶 |
| 最大堆叠 | 20 |
| 使用时间 | 2.5 秒 |
| 效果 | 清除身上大部分负面 Buff |

详见 [NPC 相关物品](npc-items.md#安神滴剂)。

## 冷淬液

重铸系统的关键材料，用于在重铸时锁定指定属性。

| 属性 | 说明 |
|------|------|
| 获取方式 | 叮当好感度 Lv.4 奖励；叮当商店；已婚叮当每日赠送 |
| 用途 | 重铸时锁定不想改变的属性 |

详见 [NPC 相关物品](npc-items.md#冷淬液) 和 [重铸系统](../systems/reforge.md)。
