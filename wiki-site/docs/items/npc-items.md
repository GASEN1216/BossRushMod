# NPC 相关物品

# 钻石

一颗真正的钻石。可以直接使用召唤叮当，也可以作为礼物送给 NPC。

- 获取方式：叮当好感度 Lv.2 解锁后购买；叮当好感度升级奖励
- 使用效果：召唤叮当跑向你，叮当好感度 +5
- 作为礼物赠送：属于"喜欢的物品"，好感度 +80（叮当/羽织通用）
- 无叮当时使用：物品仍会消耗，显示"似乎没有反应..."

# 砖石

一块轻飘飘的石头，看起来闪闪发光……但它是假的。

- 获取方式：基地商人购买（库存 5）
- 使用效果：召唤叮当跑向你，叮当好感度 -20（被识破为假钻石）
- 作为礼物赠送：叮当 -40，羽织 -60（让她想起实验室）
- 无叮当时使用：物品仍会消耗，显示"似乎没有反应..."

::: warning
把假钻石送给哥布林工匠，后果自负。
:::

---

# 钻石戒指

一枚镶嵌着璀璨钻石的精致戒指。送给好感度满级的 NPC 可以触发求婚。

- 获取方式：叮当好感度 Lv.7 后在叮当商店解锁
- 好感度加成：+500
- 求婚条件：NPC 好感度达到 Lv.10

# 求婚规则

- 好感度满级时送出戒指 → 触发婚礼
- 已婚状态下送给同一配偶 → 被拒绝，戒指不消耗
- 已婚状态下送给其他 NPC → 触发"出轨惩罚"，戒指被消耗，好感度下降

详见好感度与婚姻。

# 冷淬液

哥布林工匠重铸时使用的极寒液体，可以在重铸过程中锁定指定属性。

- 获取方式：叮当好感度 Lv.4 升级奖励；叮当商店购买；已婚叮当每日聊天赠送
- 用途：在重铸界面中锁定不想被改变的属性

这是重铸系统中最关键的材料。详见重铸系统。

# 安神滴剂

羽织亲手调配的安神滴剂，带着淡淡草药香。

- 获取方式：羽织好感度 Lv.3 一次性奖励 5 瓶
- 最大堆叠：20
- 使用时间：2.5 秒
- 效果：清除身上大部分负面 Buff

# 平安护身符

一枚承载着平安归来心意的护身符。

- 获取方式：羽织好感度 Lv.8 一次性奖励
- 最大堆叠：1
- 触发条件：生命值降至 50% 以下时，10% 概率触发
- 效果：完全回满生命值
- 限制：每个场景只能触发一次

护身符是被动效果，放在背包里即可生效，不需要主动使用。

# 叮当涂鸦

叮当花了好多天画的一幅画。使用后可以全屏查看。

- 获取方式：叮当好感度 Lv.10 特殊礼物
- 使用后消耗：否（耐久 999）
- 使用效果：打开全屏图片查看器

::: warning
注意：如果你把这幅画送回给叮当，他会非常伤心（好感度 -300）。
:::

---

# 荒野号角

一只古老的号角，吹响它可以召唤坐骑。

- 获取方式：商店或掉落
- 使用后消耗：否（耐久 999）
- 冷却时间：3 秒
- 效果：首次使用召唤坐骑，坐骑在身边时再次使用可呼唤过来

如果在配置中启用了狼模型，坐骑外观会变为狼，并且改吃饺子（而非胡萝卜）。
