# 入场与功能物品

## BossRush 船票

进入标准 BossRush、无间炼狱、白手起家和血猎追击的基础凭证。

| 属性 | 说明 |
|------|------|
| 获取方式 | 基地商人购买 |
| 库存上限 | 10 张 |
| 消耗方式 | 选择地图进入竞技场时自动消耗 |

船票是所有 BossRush 模式的入场门槛。白手起家需要裸装+船票，血猎追击需要裸装+船票+血猎收发器。

---

## 冒险家日志

游戏内百科书，使用后打开 Wiki 界面，可以查阅 Mod 的所有内容说明。

| 属性 | 说明 |
|------|------|
| 获取方式 | 基地商人购买 |
| 库存上限 | 1 本 |
| 使用后消耗 | 否，可反复使用 |

百科内容支持中文和英文，会随 Mod 版本更新。

## 成就勋章

成就系统的入口物品。右键使用可打开成就面板（也可以按 L 键直接打开）。

| 属性 | 说明 |
|------|------|
| 获取方式 | 基地商人免费领取 |
| 库存上限 | 1 个 |
| 使用后消耗 | 否 |

详见 [成就大全](../achievements/index.md)。

---

## 血猎收发器

血猎追击模式的专属入场钥匙。

| 属性 | 说明 |
|------|------|
| 获取方式 | 基地商人购买 |
| 库存上限 | 5 个 |
| 品质 | 史诗（紫色） |
| 消耗方式 | 进入血猎追击时自动消耗 |

需要与船票一起裸装携带才能触发血猎追击模式。详见 [血猎追击](../game-modes/mode-f.md)。

---

## 营旗（6 种 + 1 随机）

划地为营模式的入场凭证，决定你加入的阵营。

| 营旗 | 对应阵营 |
|------|---------|
| 随机营旗 | 随机分配到 5 个 NPC 阵营之一 |
| 拾荒者营旗 | 拾荒者 |
| USEC 营旗 | USEC |
| BEAR 营旗 | BEAR |
| 实验室营旗 | 实验室 |
| 狼群营旗 | 狼群 |
| 爷的营旗 | 独狼（与所有阵营为敌） |

| 属性 | 说明 |
|------|------|
| 获取方式 | 基地商人购买 |
| 每种库存 | 5 面 |
| 消耗方式 | 进入划地为营时自动消耗 |

详见 [划地为营](../game-modes/mode-e.md)。
