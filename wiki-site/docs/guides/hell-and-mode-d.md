# 无间炼狱与白手起家攻略

# 无间炼狱——没有终点，只有更远

Boss 每波 +2% 属性，无限叠加。你的目标就一个：**活得尽可能久**。

# 入场前准备

- 带你最好的装备，尤其是高品质武器和充足弹药
- 龙王套装的冲刺+火焰免疫在后期是**生命线**
- 腾云驾雾图腾提供空中规避
- 逆鳞兜底保命

# 战斗节奏

- **前 20 波是热身**——四骑士、龙裔遗族、焚天龙皇等猛 Boss 不会出现，适应节奏
- 路牌旁有弹药商店，随时补给
- 现金 2 米内自动飞向你，但得先靠近吸附范围
- 每 5 波掉一件高品质物品，别忘了捡

# 后期怎么活？

- 50 波以上 Boss 属性会非常恐怖——**保持移动，别站桩**
- Boss 筛选器调整权重，减少最烦的 Boss 出场率
- 里程碑奖励（每 100 波）指数增长——**坚持越久越赚**

# 成就目标

- **10 波** → 无间炼狱·十 / **10 波无伤** → 钢铁意志 / **10 波不治疗** → 铁人挑战（隐藏）
- **25 波** → 无间炼狱·廿五 / **50 波** → 五十 / **100 波** → 百 / **200 波** → 双百

# 白手起家——啥都没有，全靠捡

随机开局，装备全随机。关键是**快速适应手里的牌**。

# 开局策略

- 开局装备完全随机，别追求完美——**有什么用什么**
- 检查武器类型和弹药量，决定打法
- 拿到近战没枪？前几波近战清场也行
- 医疗品只有 3 件，**非常金贵**，尽量少挨打

# 波次攻略

- **1-2 波** — 最弱小兵，轻松清理，搜刮升级
- **3-5 波** — 小兵变强但没 Boss，重点找武器和护甲
- **6-10 波** — 出 1 个 Boss，优先击杀拿掉落
- **11-15 波** — 2 个 Boss，走位别被夹击
- **16+ 波** — 全 Boss 池开放，装备应该成型了

# 装备升级

- 敌人装备品质随波次提升，后期小兵也能掉品质 5-6 的好货
- 每波结束翻战利品箱，**优先替换最菜的装备位**
- Boss 掉落一般比小兵好

# 成就目标

- **完成 10 波** → 白手起家
- **5 波无伤** → 完美白手
