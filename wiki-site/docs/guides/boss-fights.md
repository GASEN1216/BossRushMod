# Boss 战攻略

# 龙裔遗族攻略

# 难度评估
- 中等难度。双阶段机制是核心考验，第二阶段的狂暴状态需要提前准备。

# 第一阶段要点
- 伤害倍率低（0.3x），压力不大。
- 注意每 10 发子弹附带的火箭弹，爆炸半径 1 米，保持距离。
- 每 5 秒一颗燃烧弹，注意脚下火焰区域。
- 稳定输出，不需要急。

# 第二阶段要点
- 50% 血量时会"假死"并复活，不要提前庆祝。
- 复活后伤害大幅提升（1.1x + 1.5x 暴击），追击速度 ×1.5。
- 接触即造成碰撞伤害，保持距离。
- 燃烧弹频率提升到每秒 1 颗，且会朝自身投掷来回血。
- 关键：保持高输出压制，不给它回血的机会。

# 克制手段
- 穿龙裔/龙王套装：火焰伤害反而给你回血，大幅降低难度。
- 冰属性武器/弹药：第二阶段可以减速它。
- 逆鳞图腾：万一翻车还有一次保命机会。

# 无伤击杀技巧
- 第一阶段保持中距离，躲避火箭弹和燃烧弹即可。
- 第二阶段是难点：利用冲刺拉开距离，不要让它近身。
- 穿龙裔套装可以无视火焰伤害，只需要躲避碰撞和子弹。
- 用 Boss 筛选器禁用其他 Boss，确保单独面对。

# 焚天龙皇攻略

# 难度评估
- 最高难度。7 种技能 + 双阶段 + "孩儿护我"机制，需要充分准备和多次练习。

# 通用要点
- 焚天龙皇悬浮在空中（3-5 米），近战武器很难够到，优先使用枪械。
- 它会跟随你移动，不要试图跑出它的追踪范围。
- 保持移动是最重要的生存法则。

# 技能应对
- 棱彩弹：8 颗追踪弹，追踪 2.5 秒后转直线，持续横向移动可以甩开。
- 冲锋：0.8 秒蓄力 + 地面预警圆环，看到预警立刻垂直于冲锋方向移动。
- 日轮舞：瞬移到你附近，找到旋转光束间隙站位，或向外跑出范围。
- 万彩虹环：13 颗星体螺旋扩散，扩散阶段向外跑，收缩阶段向内靠。
- 虚影枪：1 秒预警线，看清预警线方向，站在线与线之间。

# 第二阶段变化
- 攻击间隔从 1.0 秒缩短到 0.5 秒。
- 冲锋变为连续两次。
- 新增螺旋棱彩弹和虚影枪阵（4 波 ×16 根）。
- 需要更快的反应和更积极的走位。

# "孩儿护我"阶段
- 龙皇 1 血时飞升并召唤半属性的龙裔遗族。
- 龙皇每 3 秒发射棱彩弹。
- 优先击杀龙裔遗族，同时注意躲避棱彩弹。
- 龙裔遗族死后龙皇才会真正死亡。

# 推荐装备
- 龙裔/龙王套装：火焰免疫 + 冲刺机动。
- 高 DPS 枪械：缩短战斗时间 = 减少犯错机会。
- 腾云驾雾图腾：空中规避地面技能。
- 逆鳞：保命用。
- 充足弹药和医疗品。

# 无伤击杀技巧
- 需要完全熟悉所有技能的预兆和安全站位。
- 龙王套装的冲刺是最好的规避手段（0.5 秒冷却，6 米距离）。
- 第二阶段是最大挑战，攻击频率翻倍。
- "孩儿护我"阶段注意同时应对两个目标。
- 建议先用 Boss 筛选器清空其他 Boss，反复练习。

# 通用 Boss 战建议
- 利用竞技场的掩体和地形优势。
- 路牌旁的弹药商店可以随时补充弹药。
- 多 Boss 波次中，注意屏幕上的方位提示，避免被包围。
- 如果某个 Boss 特别难对付，用 Boss 筛选器暂时禁用它。
- 击杀速度影响掉落品质——打得越快，高品质掉落概率越高。
- 详细的掉落机制见掉落与奖励。

::: tip
穿龙裔/龙王套装可以大幅降低两个龙系 Boss 的难度，火焰伤害会转化为治疗。
:::
