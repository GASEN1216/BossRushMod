# 新手推荐路线

# 第一步——先打一局试试

1. 去基地商人买 **BossRush 船票**
2. 顺手领免费的 **成就勋章**，再买本 **冒险家日志**（就是你在看的这个）
3. 选 **DEMO 终极挑战** 地图（默认地图，最简单）
4. 难度选 **弹指可灭**（每波 1 个 Boss，新手友好）
5. 全灭 → 拿通关箱 → 撤离

::: tip
恭喜，"初出茅庐"和"弹指可灭"两个成就到手。
:::

# 打完第一局，然后呢？

# 提升难度
- 试试"有点意思"难度（每波 3 个），感受一下真正的压力
- 试试无间炼狱，看你能扛几波

# 去认识 NPC
- 回普通地图找 **叮当**（哥布林工匠）和 **羽织**（护士）
- 每天聊天、送礼，慢慢刷好感度
- 叮当 Lv.2 开商店、Lv.4 给冷淬液——之后就能重铸装备了

# 挑战自定义 Boss
- 标准 BossRush 里会遇到 **龙裔遗族**，先学它的双阶段机制
- 打掉它拿龙裔套装（赤龙首 + 焰鳞甲），穿上火焰免疫+冲刺
- 这套装备够你用很久

# 解锁其他模式
- **白手起家** — 裸装+船票入场，随机开局的成长挑战
- **划地为营** — 裸装+营旗入场，多阵营混战沙盒
- **血猎追击** — 裸装+船票+血猎收发器，**最难的生存模式**

# 装备成长路线

基础装备 → **龙裔套装+龙息**（打龙裔遗族）→ **龙王套装+焚皇断界戟+图腾**（打焚天龙皇）→ **重铸强化**（叮当重铸+冷淬液锁属性）

# 钱从哪来？

- **前期**：卖通关奖励箱里的物品
- **中期**：刷成就，成就奖金是最稳的大额收入
- **后期**：无间炼狱的现金池和里程碑奖励
- NPC 好感度折扣能省不少重铸和治疗费

# 常用快捷键

- **L** — 成就面板
- **Ctrl+F10** — Boss 筛选器
