# 划地为营攻略

# 核心思路

沙盒混战，没有固定胜负条件。你可以：
- 帮自己阵营干掉其他阵营
- 坐山观虎斗，等他们打完你收割
- 选独狼，**一个人打全图**

# 阵营怎么选？

- **随机** — 随机分配，喜欢惊喜的选这个
- **拾荒者 / USEC / 实验室** — 标准阵营，Boss 池均衡，常规体验
- **BEAR** — Boss 属性 x2.5，友军也更强。**想要猛队友就选它**
- **狼群** — 优先生成狼群特色 Boss，想体验狼群特色的选
- **独狼** — 没有友军，5 个阵营都是你的敌人。**你确定？**

# 战术消耗品什么时候用？

- **挑衅烟雾弹** — 场上 Boss 少了，补充新目标
- **混沌引爆器** — 想看全图大混战时扔一个，场面壮观
- **猎王响哨** — 把附近 Boss 集中过来围歼
- **血狩烽火** — 全图 Boss 追你。**高风险高回报，勇者限定**

# 经济策略

- 神秘商人弹药原价，其他商品 **x10 倍价格**
- 优先买弹药，购买后自动补满堆叠
- 击杀**敌对阵营** Boss 掉战利品箱（主要装备来源）
- 击杀同阵营 Boss 不掉箱子——别打友军

# 成长机制——越打越强

- **Boss 成长**：每只 Boss 独立结算，同阵营每有单位阵亡就叠层。每层 +5% 生命/+5% 枪伤/+5% 近战伤。新刷的 Boss 不继承旧层数
- **玩家成长**：亲手给敌对 Boss 补最后一刀就叠 1 层。每层 +0.1% 生命/枪伤/近战伤，持续累积
- 叠层时头上显示：`生命/伤害+0.1%，总加成xx%`

# 独狼攻略

- 没友军，5 个阵营全是敌人
- 开局送额外补给物资
- **核心策略：让不同阵营 Boss 互相打**，你坐收渔利
- 血狩烽火可以把全图 Boss 引到一起互相混战——**让他们自相残杀**
- 最难的模式变体，装备成型后再来
