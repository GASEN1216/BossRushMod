# 划地为营攻略

# 核心思路
- 划地为营是一个沙盒混战模式，没有固定胜负条件。
- 你的目标可以是：
- 帮助自己的阵营消灭其他阵营。
- 利用阵营间的混战坐收渔利。
- 以独狼身份挑战全图所有 Boss。

# 阵营选择建议
- **随机**：随机分配，增加不确定性，想要惊喜的玩家。
- **拾荒者 / USEC / 实验室**：标准阵营，Boss 池均衡，常规体验。
- **BEAR**：Boss 获得大幅属性加成（×2.5），友军更强，想要强力友军的玩家。
- **狼群**：优先生成所有独特狼群 Boss，想体验狼群特色的玩家。
- **独狼**：没有友军，与所有阵营为敌，硬核玩家，追求极限挑战。

# 战术消耗品使用时机
- **挑衅烟雾弹**：场上 Boss 数量减少时，补充新的战斗目标。
- **混沌引爆器**：想要制造全图大混战时使用，场面会非常壮观。
- **猎王响哨**：想要集中附近 Boss 进行围歼时。
- **血狩烽火**：想要全图 Boss 追击你时（高风险高回报）。

# 经济策略
- 神秘商人的弹药是原价，其他商品 ×10 倍价格。
- 优先购买弹药补给，弹药购买后自动补满堆叠。
- 击杀敌对阵营 Boss 会掉落战利品箱，是主要装备来源。
- 击杀同阵营 Boss 不掉落战利品。

# 成长机制的利用
- Boss 成长按个体独立结算：每只 Boss 只吃自己出生后、同阵营后续阵亡单位带来的层数。
- Boss 每层 +5% 生命、+5% 枪械伤害、+5% 近战伤害；新补刷 Boss 不继承旧层数。
- 玩家成长只看你是否亲手完成对敌对 Boss 的最后一击，每次完成最后一击获得 1 层。
- 玩家每层仅 +0.1% 生命、+0.1% 枪械伤害、+0.1% 近战伤害，但可持续累积。
- 每次玩家叠层会在头上显示：`生命/伤害+0.1%，总加成xx%`。

# 独狼模式攻略
- 独狼没有友军，所有 5 个阵营的 Boss 都是你的敌人。
- 开局会额外获得补给物资。
- 利用不同阵营 Boss 之间的互相攻击来消耗它们。
- 血狩烽火可以把全图 Boss 引到一起互相混战。
- 这是最难的模式变体，建议装备成型后再尝试。
