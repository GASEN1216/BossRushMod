# 血猎追击攻略

# 核心思路

Mod 里最难的模式。**你在持续失血，杀 Boss 才能续命。** 四个阶段失血越来越快，最终目标是活着撤离。

# 准备阶段（180 秒，1%/秒）

最轻松的窗口期，抓紧：
- 尽快击杀几个 Boss 回血 + 拿工事道具
- 搜刮战利品箱升级装备
- 在关键位置预部署掩体
- 熟悉地图，记住刷怪点

# 悬赏阶段（180 秒，1.5%/秒）

悬赏名单出了，**所有存活 Boss 被标记**：
- 关注悬赏雷达，**优先杀高印记 Boss**
- 悬赏 Boss 回血更多（75% vs 普通的 50%），额外掉落也更好
- Boss 之间也会互杀并继承印记——留意悬赏榜首变化
- 印记越多 = 撤离奖励越丰厚

# 猎潮阶段（180 秒，2%/秒）

**全面追杀开始。**
- 所有 Boss 冲你来，未标记 Boss 移速 +50%
- 最危险的战斗阶段——**必须保持高击杀节奏维持血量**
- 工事在这个阶段是命，用掩体和路障给自己创造安全输出位
- 别停下来，持续移动 + 战斗

# 撤离阶段（无限，3%/秒）

- 撤离点生成在离你很远的地方
- Boss 移速 +50%，未标记 +100%
- **每秒 3% 失血，犹豫就是死**
- 别恋战，直奔撤离点
- 撤离点站 15 秒完成撤离
- 跑路途中可以边跑边部署工事

::: warning
撤离阶段每秒 3% 失血。不跑 = 死。
:::

# 工事怎么用？

- **折叠掩体** — 随时扔，提供临时掩护，数量最多
- **加固路障** — 堵关键通道或撤离点，5000 血很耐打
- **铁丝网** — 铺在 Boss 必经之路，拖延推进速度
- **维修喷剂** — 修受损工事，延长寿命

# 撤离点防御——最后 15 秒

1. 到撤离点后**立刻部署工事**
2. 加固路障堵主要方向
3. 铁丝网减速接近的 Boss
4. 折叠掩体补缺口
5. 维修喷剂维护受损工事
6. **坚持 15 秒**

# 悬赏印记——要不要贪？

- 印记越多奖励越丰厚，但更多战斗 = 更大风险
- **活着撤离比多几个印记重要得多**
- 高印记 Boss 回血更多（75% + 印记数的最大生命值增长），该杀还是杀

# 推荐装备

- **高 DPS 武器** — 杀 Boss 速度 = 续命速度
- **龙王套装** — 冲刺机动 + 火焰免疫
- **腾云驾雾** — 飞着赶路到撤离点
- **逆鳞** — 最后的保命手段
- 弹药备足，神秘商人可以买

::: tip
血猎追击最核心的是节奏感——什么时候打、什么时候跑、什么时候部署工事。节奏对了就能活。
:::
