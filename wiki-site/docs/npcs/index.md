# NPC 总览

BossRush Mod 添加了 3 位 NPC，各自提供不同的服务。他们出现在基地场景中，部分 NPC 也会在特定模式内出现。

# NPC 列表
- 叮当：哥布林工匠，主要服务为重铸、商店、聊天
- 羽织：护士，主要服务为治疗、清除 Debuff
- 阿稳：快递员，主要服务为物品寄存与取回

# 好感度系统

叮当和羽织拥有完整的好感度系统：

- 10 个等级，通过每日聊天和送礼提升
- 每个等级解锁新的服务、折扣或奖励
- 好感度满级后可以用钻石戒指求婚
- 连续不互动会导致好感度缓慢衰减

详见好感度与婚姻。

# NPC 出现位置
- 基地：叮当 ✓ / 羽织 ✓ / 阿稳 ✓
- 婚礼教堂（已婚后）：叮当 ✓ / 羽织 ✓ / 阿稳 —
- 划地为营（模式内）：叮当 — / 羽织 — / 阿稳 ✓
- 血猎追击（模式内）：叮当 — / 羽织 — / 阿稳 ✓

划地为营和血猎追击模式内还会出现神秘商人，提供分类商店服务。
