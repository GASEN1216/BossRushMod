# NPC 总览

BossRush 里有三个原创 NPC 陪你闯荡——一个帮你打铁，一个给你治病，一个帮你搬家。不是工具人，是真有感情戏的那种。

# 三位伙伴

- **叮当**（哥布林工匠）— 帮你重铸装备、开商店、聊人生。装备养成线的核心人物
- **羽织**（护士）— 给你治伤、清 Debuff。温柔但收费不温柔
- **阿稳**（快递员）— 帮你存东西、取东西、跨场景搬家。全勤劳模

# 好感度——日久生情

叮当和羽织有完整的好感度系统：

- 10 个等级，每天聊聊天、送送礼就能涨
- 每个等级解锁新服务、折扣或剧情
- 满级之后可以拿钻石戒指求婚（没错，真的能结婚）
- 几天不理他们好感度会慢慢掉，**养 NPC 跟养花一样得浇水**

详见好感度与婚姻。

# 在哪能找到他们？

- **普通地图** — 三位全在
- **婚礼教堂**（结婚后）— 配偶在，阿稳不在
- **标准 BossRush / 白手起家** — 叮当和羽织随机出一个（已婚的不参与随机），阿稳必出
- **划地为营 / 血猎追击** — 三位全员到场

划地为营和血猎追击里还会多出一个神秘商人，专门卖军火补给。
