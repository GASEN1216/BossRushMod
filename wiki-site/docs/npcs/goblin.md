# 叮当（哥布林工匠）

# 他是谁？

一只由 J-Lab 实验室创造出来的、聪明得不像哥布林的哥布林。在 BossRush 里他是你的装备养成核心——想重铸装备、想买好东西，都得找他。对他好点，他会回报你的。

# 他能帮你啥？

- **聊天**（初始就有）— 每天聊一次，好感度 +40。天天聊就完了
- **送礼**（初始就有）— 每天送一件，喜欢的东西加更多分
- **商店**（好感度 Lv.2 解锁）— 卖钻石、冷淬液等好东西，越熟越打折
- **重铸**（初始就有）— 装备属性不满意？他帮你重洗

# 好感度升上去有什么好处？

- **Lv.2** — 商店开放，送你一颗钻石
- **Lv.3** — 商店打 9 折
- **Lv.4** — 送你冷淬液（重铸必备）
- **Lv.5** — 解锁剧情对话：叮当的过去·上
- **Lv.6** — 商店打 85 折
- **Lv.7** — 商店上架钻石戒指（求婚用的）
- **Lv.10** — 剧情对话：叮当的过去·下；商店打 8 折；获赠叮当涂鸦

# 送什么他开心？

- **喜欢**：钻石、生日蛋糕、配方/蓝图类（+80）
- **特别喜欢**：钻石戒指（+500，不过送了就是求婚了）
- **普通**：其他随便什么（+20）
- **讨厌**：砖石/假钻石（-40，他不蠢，看得出来）

::: warning
千万别把叮当涂鸦送回给他——好感度直接 -300。那幅画对他意义非凡。
:::

# 重铸服务

叮当的重铸是装备养成的核心。好感度越高，重铸费越便宜：Lv.3 九折，Lv.6 八五折，Lv.10 八折。详见重铸系统。

# 可以结婚吗？

可以。好感度 Lv.10 + 送出钻石戒指 = 和叮当结婚。婚后他搬到婚礼教堂，每天聊天还额外送你冷淬液——**嫁给工匠的好处就是永远不缺耗材**。

详见好感度与婚姻。
