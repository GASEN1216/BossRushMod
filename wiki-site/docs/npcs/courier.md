# 阿稳（快递员）

# 他是谁？

你的私人快递小哥——不管你在基地还是在竞技场，他永远在那里等着帮你搬东西。没有好感度系统，不需要你每天聊天送礼，**纯粹的工具人，而且他乐在其中**。

# 他能帮你啥？

- **快递服务** — 把物品扔进快递箱，付运费送回家
- **一键存入** — 背包里的东西一键全塞进快递箱（锁定格自动跳过）
- **物品寄存** — 存入寄存点，跨场景保存，换地图也不丢
- **一键寄存** — 背包物品一键全存进寄存点
- **物品取回** — 从寄存点拿回你之前存的东西

# 寄存点——你的跨场景仓库

- 数据持久化保存，关游戏重启都不会丢
- 只要有阿稳在的地方就能取回——他就是你的移动保管箱
- 不同模式之间转移物品就靠这个

# 血猎追击联动

血猎追击成功撤离后，悬赏奖励会**自动发送到寄存点**。回普通地图找阿稳取就行，方便得很。

# 阿稳快递牌

- 基地商人处有卖
- 用了之后穿戴、背包、宠物背包里的东西**一键快递回家**
- 在基地用免费，其他地方按正常快递收费

# 阿稳扫箱令

- 划地为营/血猎追击中每累计 20 次 Boss 死亡自动获得 1 个
- 用了之后阿稳帮你按由近到远清理当前已有的掉落箱——**满地箱子看得头疼的时候就靠它了**
- 只清使用时已存在的箱子，扫箱过程中不能重复使用

# 在哪能找到他？

**到处都有。** 普通地图、标准 BossRush、白手起家、划地为营、血猎追击——全勤。

# 实用贴士

- 没有好感度系统，不用养，随用随找
- 进高风险模式前先把贵重装备存到阿稳那里，别带进去赔了
- 清背包用一键存入/一键寄存，锁定格不会被误操作
- 血猎撤离奖励在寄存点等你，**别忘了去取**
- 阿稳在所有 BossRush 模式内都在，随时可以存取
