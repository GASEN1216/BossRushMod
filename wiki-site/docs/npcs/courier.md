# 阿稳（快递员）

# 概述

阿稳是一位快递员 NPC，提供物品寄存和取回服务。他出现在基地场景中，也会在所有 BossRush 模式内出现，方便你在战斗中随时存取物品。

# 服务
- 物品寄存：将物品存入寄存点，跨场景保存
- 物品取回：从寄存点取回之前存入的物品

# 寄存点机制
- 寄存点的数据会持久化保存，不会因退出游戏而丢失
- 存入的物品可以在任何有阿稳出现的场景中取回
- 寄存点是你在不同模式之间转移物品的桥梁

# 与血猎追击的联动

血猎追击模式成功撤离时，悬赏奖励物品会自动发送到寄存点。撤离后回到基地找阿稳即可取回这些奖励。

# 出现位置
- 基地：✓
- 标准 BossRush（模式内）：✓
- 白手起家（模式内）：✓
- 划地为营（模式内）：✓
- 血猎追击（模式内）：✓

# 使用建议
- 阿稳没有好感度系统，不需要每日互动，随时找他存取物品即可
- 在进入高风险模式前，可以把贵重物品存到阿稳处，避免意外损失
- 血猎追击的撤离奖励会自动发送到寄存点，记得撤离后去取
- 阿稳在所有 BossRush 模式内都会出现，可以随时存取装备
