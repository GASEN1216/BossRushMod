# 羽织（护士）

# 概述

羽织是前 J-Lab 实验室的高级医疗研究员，实验室崩溃后逃到 Duckov 开设了野战诊所。她提供治疗服务，并拥有独立的好感度和婚姻线。

# 服务
- 聊天（初始）：每日聊天可提升好感度（+30/天）
- 送礼（初始）：每日可送一件礼物
- 治疗（初始）：回复生命值并清除负面 Buff，费用随好感度打折

# 好感度等级奖励
- Lv.2：治疗 10% 折扣
- Lv.3：获得安神滴剂 ×5
- Lv.4：治疗 20% 折扣
- Lv.5：剧情对话（羽织的回忆）
- Lv.6：治疗 25% 折扣
- Lv.7：治疗 30% 折扣
- Lv.8：获得平安护身符；聊天时出现爱心气泡
- Lv.9：治疗 40% 折扣
- Lv.10：剧情对话（羽织的告白）

# 送礼偏好
- 喜欢：钻石、生日蛋糕、消耗品/医疗品类物品（+80）
- 特别喜欢：钻石戒指（+500）
- 普通：其他物品（+20）
- 讨厌：砖石（让她想起实验室）（-60）

# 婚姻

好感度达到 Lv.10 后，送出钻石戒指可以与羽织结婚。婚后羽织会搬到婚礼教堂，对话内容变为婚后专属版本。

详见好感度与婚姻。

# 小贴士
- 羽织的治疗折扣最终可以达到 40%，长期互动收益很高
- 安神滴剂可以清除大部分负面 Buff，在高难度模式中非常实用
