# 成就大全

# 概述
BossRush Mod 当前共有 35 个成就，分为 7 个分类。完成成就可获得现金奖励。按 L 键（默认，可在配置中修改）打开成就面板查看进度。

# 基础通关
- 初出茅庐：首次在 BossRush 中成功撤离。奖励 `$5,000`，难度 `★`
- 弹指可灭：在[弹指可灭]难度成功撤离。奖励 `$10,000`，难度 `★`
- 有点意思：在[有点意思]难度成功撤离。奖励 `$25,000`，难度 `★★`
- 白手起家：在白手起家模式完成 10 波。奖励 `$30,000`，难度 `★★`
- 无间炼狱·十：在无间炼狱模式完成 10 波。奖励 `$50,000`，难度 `★★★`
- 无间炼狱·廿五：在无间炼狱模式完成 25 波。奖励 `$150,000`，难度 `★★★`
- 无间炼狱·五十：在无间炼狱模式完成 50 波。奖励 `$350,000`，难度 `★★★★`
- 无间炼狱·百：在无间炼狱模式完成 100 波。奖励 `$800,000`，难度 `★★★★★`
- 无间炼狱·双百：在无间炼狱模式完成 200 波。奖励 `$2,000,000`，难度 `★★★★★`
