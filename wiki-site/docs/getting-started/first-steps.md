# 新手上路

# 第一步：搞张船票

去基地商人那里，买一张 **BossRush 船票**——这是你踏入 Boss 竞技场的入场券，没它哪儿也去不了。

顺手再拿两样东西：

- **冒险家日志** — 就是你现在在看的这本百科书，随时翻
- **成就勋章** — 免费的，白拿不拿是傻子。右键打开成就面板

# 第二步：挑张地图

有了船票，地图选择界面会解锁 9 张 BossRush 地图。第一次？直接选 **DEMO 终极挑战**，场地又大又平，不会迷路。

# 第三步：选个难度

落地之后，你会看到场地中央有块路牌。走过去互动，选难度：

- **弹指可灭** — 每波 1 个 Boss，适合试水，一个一个慢慢揍
- **有点意思** — 每波 3 个 Boss 一起上，节奏感拉满
- **无间炼狱** — 无限波次，Boss 越打越强，看你能撑几轮

# 第四步：开打！

- 标准难度下每波 Boss 死了会掉箱子，里面全是好东西；无间炼狱不掉箱，改发现金和里程碑大奖
- 波次之间有休息时间，够你翻翻箱子、补补弹药
- 路牌旁边有弹药商店、维修站和垃圾桶——文明竞技，别乱扔箱子

# 打完之后呢？

标准 BossRush 觉得不够刺激了？试试这些：

- **白手起家** — 裸装 + 船票，系统随机给你发装备，拿到什么用什么
- **划地为营** — 裸装 + 营旗，选个阵营，满图 Boss 大乱斗
- **血猎追击** — 裸装 + 船票 + 血猎收发器，进去就开始掉血，不杀 Boss 就等死
- **无间炼狱** — 选无间炼狱难度，看看自己的极限在哪

# 新手锦囊

::: tip
路牌上有个"返回出生点"，迷路了或者被打残了赶紧用。
:::

:::
