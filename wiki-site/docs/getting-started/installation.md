# 安装与启用

# 通过 Steam 创意工坊安装

- 1. 在 Steam 中打开 Escape from Duckov 的创意工坊页面
- 2. 搜索 "BossRush" 或直接访问 Mod 页面
- 3. 点击"订阅"
- 4. 启动游戏，Mod 会自动加载

# 确认 Mod 已生效

进入游戏后，你可以通过以下方式确认 Mod 正常运行：

- 基地商人处出现 **BossRush 船票** 等新物品在售
- 按 **L 键**（默认）可以打开成就面板
- 普通地图场景中可以看到 NPC **叮当**（哥布林）和 **羽织**（护士）

# 配置文件

Mod 首次运行时会自动在 StreamingAssets/ 目录下生成配置文件 BossRushModConfig.txt。你可以通过编辑该文件来调整各项参数，详见配置选项。

如果游戏支持 ModConfig UI，也可以在游戏内的 Mod 设置界面中直接修改。
