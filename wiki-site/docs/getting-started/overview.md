# Mod 简介

# 所以……BossRush Mod 到底是啥？

简单来说：一个把 Escape from Duckov 变成 Boss 竞技场的大型 Mod。从创意工坊一键订阅，进去就是干。

但如果只是打 Boss 也太无聊了——所以我们塞了 5 种模式、3 个原创 Boss、一堆 NPC 和装备养成线进去。你可以理解为：**在鸭游里开了个主题乐园，只不过游乐设施全是 Boss。**

# 都有些啥？

- **5 种模式** — 经典波次、无尽生存、白手起家、多阵营混战、持续失血的血猎追击……总有一款让你怀疑人生
- **9 张地图** — 从雪地到迷宫，全图可选
- **3 位原创 Boss** — 龙裔遗族（会复活的龙崽）、焚天龙皇（会喊儿子来护驾的终极龙王）、幽灵女巫（隐身闪现一套带走你）
- **3 位 NPC 伙伴** — 叮当（能帮你重铸装备的哥布林工匠）、羽织（温柔但账单不温柔的护士）、阿稳（随叫随到的快递小哥）
- **9 件原创装备** — 从火龙套到冰系大剑再到幽灵镰刀，装备线比你的购物车还长
- **20+ 自定义物品** — 船票、营旗、工事、消耗品……别嫌多，你都用得上
- **基地建筑** — 婚礼教堂（是的，你可以和 NPC 结婚）、许愿台（许愿还能抽奖）
- **重铸系统** — 洗属性 + 锁词条，让你的装备从能用变成好用
- **好感度 & 婚姻** — 每天聊天送礼，感情到位了就求婚，婚后还有专属福利
- **35 项成就** — 通关、无伤、速通、击杀、收集……够肝一阵的
- **游戏内百科** — 你正在看的就是这个，随时翻阅
- **自由配置** — 波次间隔、Boss 数量、难度倍率全都能调

# 语言支持

中文、英文全覆盖——物品描述、NPC 对话、UI 文本、游戏内百科全部双语。不存在"看不懂"这个借口。

# 兼容性

- Steam 创意工坊一键订阅，不折腾
- 不动游戏原文件，Harmony 补丁注入，干净利落
- 跟着游戏版本同步更新，不用担心炸档
