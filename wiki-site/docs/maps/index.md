# 地图总览

BossRush Mod 支持 9 张地图，覆盖游戏内大部分场景。所有地图均可用于标准 BossRush、无间炼狱、白手起家、划地为营和血猎追击。

# 地图列表

- **DEMO 终极挑战**（DEMO Ultimate Challenge）：默认地图，场地简单开阔，适合新手
- **零度挑战**（Zero Challenge）：雪地地图，入场自动发放防寒装备
- **零号区**（Ground Zero）
- **仓库区**（Hidden Warehouse）
- **农场镇**（Farm Town）
- **J-Lab 实验室**（J-Lab Laboratory）
- **口口场地**（Underground Arena）
- **37 号实验区**（Zone 37 Experimental Area）
- **迷宫**（Maze）：复杂地形，视野受限

# 地图选择方式

- 1. 持有 BossRush 船票
- 2. 打开地图选择界面
- 3. 选择目标地图进入

# 雪地地图特殊规则

在零度挑战等寒冷地图上，白手起家、划地为营和血猎追击模式会额外发放防寒装备（头盔 + 护甲），防止玩家因寒冷而死亡。

# 选图建议

- 新手入门：DEMO 终极挑战
- 练习 Boss 战：场地开阔的地图（DEMO、农场镇）
- 划地为营混战：刷怪点多且分散的地图
- 血猎追击：有掩体和复杂地形的地图
- 追求速通成就：熟悉的小型地图
