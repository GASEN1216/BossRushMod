# Cloud Rider Totem

# Overview
- Cloud Rider is a flight Totem equipped in the Totem slot
- Once equipped, you can truly fly and Glide, completely changing how you utilize space

# Basic Info
- Type: Totem
- Quality: 6 (Legendary)
- Drop source: Skyburner Dragon Lord (20% drop rate)
- Death drop: Will not drop on death

# Flight Controls
- Hold sprint key (default Space): Fly upward
- Release sprint key: Stop ascending
- Use directional keys while flying: Control horizontal movement direction

::: warning
Flying replaces the normal sprint function.
:::

# Flight Parameters
- Max ascend speed: 10 units/sec
- Acceleration time: 0.3 sec
- Glide horizontal speed: 0.8x normal move speed
- Glide descent speed: -0.8 units/sec

# Stamina Consumption
- Takeoff: 5 Stamina
- While flying: 50/sec
- While gliding: 30/sec

::: tip
When Stamina is depleted, you automatically switch to Glide mode (slow descent). Gliding still consumes Stamina but at a lower rate.
:::

# Visual Effects
- Cloud and mist effects appear beneath your feet while flying

# Use Cases
- Rapid movement and scouting on large maps
- Gaining height advantage in complex terrain
- Boss skill evasion (fly up to dodge ground attacks)
- Overlooking the battlefield in Faction War to observe each faction's situation

# Related Achievement
- Ride the Wind — Fly for the first time using the Cloud Rider Totem (reward: $50,000)
