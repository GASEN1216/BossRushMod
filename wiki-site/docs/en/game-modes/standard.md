# Standard BossRush

# Overview

Standard BossRush is the most basic game mode. Buy a ticket, choose a map, enter the arena, then select a difficulty at the signpost to start the challenge. Defeat all waves of Bosses to clear the stage and evacuate.

# Entry Requirements

- Have a BossRush Ticket (purchased from the base merchant)

# Difficulty Selection

After entering the arena, interact with the central signpost to choose a difficulty:

- Easy: Bosses per wave 1 / Waves Limited / Clear condition Evacuate after defeating all Bosses
- Normal: Bosses per wave 3 / Waves Limited / Clear condition Evacuate after defeating all Bosses
- Infinite Hell: See Infinite Hell mode / Waves Unlimited / Cannot clear, survive until death

# Combat Flow

- 1. After selecting a difficulty, the first wave of Bosses spawns
- 2. Kill all Bosses in the current wave
- 3. Loot crates spawn at Boss death locations
- 4. The next wave starts automatically after a countdown (default 15 seconds), or can be started manually via the signpost (must be enabled in config)
- 5. Directional hints for Bosses relative to your position are displayed on screen (East/West/South/North, etc.)
- 6. In multi-Boss waves, all Bosses will surround you from different directions

# Boss Pool

- Includes all in-game Boss presets
- The strongest elite Bosses (Four Horsemen series) will not appear in the first 10 waves, giving you warm-up time
- You can use the Boss Filter to disable specific Bosses

# Clear Rewards

After defeating all waves:

- Easy: Clear reward crate contains 3 high-quality items
- Normal: Clear reward crate contains 10 high-quality items

After clearing, the arena opens an evacuation point. Interact with it to return to base.

# Arena Facilities

The following facilities are available around the signpost:

- Ammo Shop — Purchase various ammo supplies
- Repair Station — Repair equipment durability
- Trash Can — Clean up loot crates on the field (option to clear boxes or clear all)
- Return to Spawn — Quick teleport back to the safe zone

# Arena NPCs

The following NPCs appear inside the arena:

- Awen (Courier): Always present, provides item storage and retrieval services
- One of Dingdang or Yuori (random): Provides Reforge/shop or healing services (married NPCs are excluded from the random pool)

# Related Achievements

- Rookie — First time clearing any difficulty
- Easy — Clear Easy difficulty
- Normal — Clear Normal difficulty
- Perfect Easy — Clear Easy with no damage taken
- Perfect Normal — Clear Normal with no damage taken
- Blitz / Speed Clear / Time Assassin — Speedrun achievements

See the full achievement list for details.
