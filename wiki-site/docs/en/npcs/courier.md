# Awen (Courier)

# Who Is He?

Your personal item courier. No Affinity, no gifts, no drama — just business. Shows up in normal maps and **every BossRush mode**.

# Services

- **Courier Service** — Move items to the courier box + pay delivery fee to ship home
- **`Store All`** — One-click: all root items → courier box (skips locked slots)
- **Storage Service** — Store items directly to the Storage Point (persistent, cross-scene)
- **`Deposit All`** — One-click: root items → Storage Point (skips locked)
- **Item Retrieval** — Grab your stored items back

# Storage Point

- Data persists across game exits — nothing gets lost
- Retrieve from any scene where Awen appears
- **Cross-mode bridge**: store items in one mode, retrieve in another

# Blood Hunt Integration

Successful extraction rewards are auto-sent to Storage Point. Find Awen on normal maps to collect them.

# Awen Courier Token

- Sold by the base merchant
- Ships all equipped + backpack + pet inventory home in one action
- Free at base; elsewhere costs standard courier fee

# Awen Loot Sweep Token

- Auto-granted: 1 per 20 Boss deaths in Faction War / Blood Hunt
- Awen sweeps all existing lootboxes from nearest to farthest
- Only works in Faction War / Blood Hunt with Awen present and lootboxes on the field

# Locations

Present in: Normal Maps ✓ / Standard ✓ / From Scratch ✓ / Faction War ✓ / Blood Hunt ✓

# Tips

- No Affinity needed — just use him whenever
- **Stash valuables before risky modes** to avoid loss
- `Store All` / `Deposit All` for fast inventory clearing
- Blood Hunt rewards auto-go to Storage — **remember to pick them up**
