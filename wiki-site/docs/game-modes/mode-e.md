# 划地为营

# 概述

划地为营是一个多阵营沙盒混战模式。你选择一个阵营加入，场上所有 Boss 分属不同阵营，同阵营互不伤害，不同阵营互相厮杀。没有固定波次，没有通关条件——这是 BossRush 里最自由、最混乱的模式。

# 入场条件

- 裸装：不能穿戴任何装备
- 携带一面营旗（从基地商人处购买）
- 营旗在入场时会被消耗

# 阵营选择

通过携带不同的营旗来决定你的阵营：

- 随机营旗：加入阵营 随机分配 / 随机加入 5 个 NPC 阵营之一
- 拾荒者营旗：加入阵营 拾荒者 (Scav)
- USEC 营旗：加入阵营 USEC
- BEAR 营旗：加入阵营 BEAR / 因原版无 BEAR 预设，BEAR 阵营单位属性提升 150%（×2.5 倍）作为补偿
- 实验室营旗：加入阵营 实验室 (Lab)
- 狼群营旗：加入阵营 狼群 (Wolf) / 优先生成所有独特狼群 Boss，剩余刷怪点出小怪提升为 Boss
- 爷的营旗：加入阵营 独狼 / 不加入任何阵营，与所有 Boss 为敌

# 核心机制

# 阵营友伤规则

- 同阵营单位之间不会互相造成伤害
- 不同阵营之间自动敌对
- 独狼玩家与所有阵营为敌，没有友军

# 界面识别

- Mode E 中角色血条名字会追加阵营后缀，方便快速判断敌我。
- 玩家自己的血条也会显示名字与当前阵营。

# 成长机制

- Boss 成长按个体独立记录：每只 Boss 只统计自己出生后、同阵营后续阵亡单位带来的层数
- Boss 每层 +5% 生命值、+5% 枪械伤害、+5% 近战伤害
- Boss 血量上限提升时会回复等量生命值
- 新补刷 Boss 不继承旧层数，会从 0 层重新开始统计
- 玩家成长仅在玩家亲手完成对敌对 Boss 的最后一击时触发，每次获得 1 层
- 玩家每层 +0.1% 生命、+0.1% 枪械伤害、+0.1% 近战伤害
- 每次玩家获得成长时，会在玩家头上显示提示气泡：`生命/伤害+0.1%，总加成xx%`

# 初始装备

- 与白手起家相同的随机初始装备
- 雪地地图额外发放防寒装备（头盔 + 护甲）
- 独狼玩家额外获得补给物资

# Boss 生成

- 所有 Boss 在模式开始时一次性全部生成
- 每个阵营的刷怪点按距离分配，你的阵营获得最近的刷怪点
- 龙裔遗族每局最多出现 1 只
- 焚天龙皇不会在划地为营中出现

# 竞技场 NPC

划地为营模式中三位 NPC 全部出现：

- 阿稳（快递员）：提供物品寄存和取回服务
- 叮当（哥布林工匠）：提供重铸和商店服务
- 羽织（护士）：提供治疗服务

# 神秘商人

模式开始时，一位神秘商人会出现在你面前，提供 13 个分类商店：

- 枪械：所有枪械 / 价格倍率 ×10
- 近战：近战武器 / 价格倍率 ×10
- 配件：枪械配件 / 价格倍率 ×10
- 弹药：所有弹药（购买自动补满） / 价格倍率 ×1（原价）
- 头盔：所有头盔 / 价格倍率 ×10
- 护甲：所有护甲 / 价格倍率 ×10
- 背包：所有背包 / 价格倍率 ×10
- 图腾：所有图腾 / 价格倍率 ×10
- 面具/耳机：所有面具与耳机 / 价格倍率 ×10
- 医疗：医疗品和注射器 / 价格倍率 ×10
- 食物：食物类物品 / 价格倍率 ×10
- 诱饵：诱饵类物品 / 价格倍率 ×10
- 其他：战术消耗品等 / 价格倍率 ×1

商人与你同阵营，拥有极高生命值，基本不会被击杀。

# 召唤煤球

与神秘商人的主要互动选项可以召唤"煤球"——一个加入你阵营的宠物 NPC 战斗伙伴。煤球会跟随你并参与战斗，帮助你对抗敌对阵营的 Boss。

# 战术消耗品

划地为营提供 4 种专属消耗品（可从神秘商人的"其他"商店购买）：

- 挑衅烟雾弹：在最近的 10 个刷怪点生成随机阵营 Boss
- 混沌引爆器：在全图所有刷怪点生成随机阵营 Boss
- 猎王响哨：强制 50 米内所有敌对 Boss 追击你
- 血狩烽火：强制全图所有敌对 Boss 追击你

每击杀 10 个 Boss，系统自动赠送 1 个挑衅烟雾弹。

# 结束条件

划地为营没有明确的胜负条件，是一个开放式沙盒模式。离开竞技场场景即结束。

# 策略提示

- 选择阵营时考虑 Boss 池的强度分布，不同阵营的 Boss 实力差异明显
- 独狼模式是最硬核的选择——你没有任何友军，所有 Boss 都是敌人
- 善用挑衅烟雾弹和混沌引爆器制造混乱，让不同阵营的 Boss 互相消耗
- Boss 成长是按个体独立累计，新补刷 Boss 不会继承旧层数
- 玩家亲手完成对敌对 Boss 的最后一击可稳定叠加玩家成长
- 详细攻略见划地为营攻略
