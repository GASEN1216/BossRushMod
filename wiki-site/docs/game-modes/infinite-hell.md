# 无间炼狱

## 概述

无间炼狱是标准 BossRush 的无尽生存变体。Boss 每波都会变强，没有终点，没有撤离——你能撑多少波？

## 入场条件

- 持有 **BossRush 船票**
- 在竞技场路牌上选择"无间炼狱"难度

## 核心规则

| 项目 | 说明 |
|------|------|
| 每波 Boss 数 | 默认 3（可在配置中调整） |
| 波次上限 | 无限 |
| Boss 强化 | 每波 +2% 生命值和伤害（累加） |
| 战利品箱 | 不掉落 |
| 经济系统 | 击杀 Boss 累积现金池，显示在路牌上 |
| 通关条件 | 无法通关，坚持到死亡为止 |

## 现金池与奖励

无间炼狱不掉落战利品箱，取而代之的是现金池系统：

- 每击杀一个 Boss，其价值会加入现金池
- 现金池总额显示在路牌上
- 场地内 2 米范围内的现金拾取物会自动飞向你

### 里程碑奖励

| 波次 | 奖励 |
|------|------|
| 每 5 波 | 1 件高品质随机物品（品质 ≥ 5，价值 ≥ 10000） |
| 每 100 波 | 里程碑大奖：皇冠 + 大量现金（数量随里程碑次数指数增长） |

100 波里程碑奖励按指数递增：

| 里程碑 | 皇冠数 | 现金 |
|--------|--------|------|
| 第 1 次（100 波） | 1 | 1000 万 |
| 第 2 次（200 波） | 2 | 2000 万 |
| 第 3 次（300 波） | 4 | 4000 万 |
| 第 4 次（400 波） | 8 | 8000 万 |
| 以此类推... | ×2 | ×2 |

## 路牌功能

无间炼狱模式下，路牌额外提供：

- 弹药补充商店
- 当前现金池总额显示
- 当前波次显示

## 策略提示

- Boss 每波 +2% 属性是累加的，到后期会非常恐怖。前期尽量保持高效击杀节奏
- 没有战利品箱意味着你需要带足装备入场，或者依赖里程碑奖励
- 利用 [Boss 筛选器](../systems/boss-filter.md) 调整 Boss 池权重，可以优化你的生存策略
- 详细攻略见 [无间炼狱与白手起家攻略](../guides/hell-and-mode-d.md)
