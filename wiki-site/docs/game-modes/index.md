# 游戏模式总览

六种模式，六种死法。从休闲打 Boss 到被 Boss 追着打，再到独立的丧尸生存，总有一款适合你。

# 一张表看明白

- **标准 BossRush** — 带张船票就能进 / 有限波次 / 打完一波换一波，经典竞技场体验 / 难度 ★★☆☆☆
- **无间炼狱** — 船票 + 选无间炼狱难度 / 无限波次 / Boss 越来越强，活到哪算哪 / 难度 ★★★★☆
- **白手起家** — 裸装 + 船票 / 无限波次 / 随机发装备，从捡垃圾开始逆袭 / 难度 ★★★☆☆
- **划地为营** — 裸装 + 营旗 / 没有波次（沙盒） / 选阵营、拉帮派、Boss 大乱斗 / 难度 ★★★☆☆
- **血猎追击** — 裸装 + 船票 + 血猎收发器 / 四个阶段 / 进去就开始掉血，不杀就死 / 难度 ★★★★★
- **末日丧尸模式** — 尸潮邀请函 / Roguelite 生存 / 裸装入场、无尽尸潮 + 污染递增 + 净化点数经济，独立于以上五种模式 / 难度 ★★★★☆

# 身上东西太多？系统帮你选

同时带了好几种入场道具？别慌，系统会按优先级帮你判定进哪个模式：

- 1. **划地为营** — 背包里有营旗？那就是划地为营
- 2. **血猎追击** — 有血猎收发器 + 船票？欢迎来到地狱
- 3. **白手起家** — 裸装 + 只有船票？白手起家走起
- 4. **标准 BossRush** — 啥都没带就一张船票？经典模式伺候

# 通用规则

- 9 张地图随便挑，每种模式都能打
- 竞技场里有弹药商店、维修站和垃圾桶——基础设施齐全，不至于弹尽粮绝
- 标准 BossRush、白手起家和血猎追击杀 Boss 掉箱子；无间炼狱改发现金；划地为营只有打敌对阵营才掉
- 末日丧尸模式是**独立的生存模式**：不走船票入场，用净化点数自成一套经济，具体规则见「末日丧尸模式」页面

::: tip
除末日丧尸模式外，其余五种模式在开局时都会随机抽取 1~3 个「变异词条」，改变本局的敌人/玩家/环境规则。详见「变异词条系统」页面。
:::
