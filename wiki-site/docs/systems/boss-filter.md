# Boss 筛选器

# 概述
- Boss 筛选器允许你自定义 Boss 池，禁用不想遇到的 Boss 或调整特定 Boss 在无间炼狱中的出现权重。
- 筛选器会跟随当前 Boss 预设列表刷新，Mode E / Mode F 运行时临时生成的小怪预设不会混进 Boss 列表。

# 打开方式
- 按 **Ctrl+F10** 打开 Boss 筛选器面板。

# 功能

# 禁用 Boss
- 在面板中取消勾选某个 Boss，它将不再出现在任何模式的 Boss 池中。
- 适用场景：
  - 想专门练习某个 Boss 时，禁用其他所有 Boss
  - 某个 Boss 太难或太烦，暂时排除
  - 追求特定 Boss 的掉落时，缩小 Boss 池提高效率

# 无间炼狱权重
- 可以为每个 Boss 单独设置在无间炼狱中的出现权重倍率：
  - 默认权重为 1.0
  - 设为 2.0 = 出现概率翻倍
  - 设为 0.5 = 出现概率减半
  - 设为 0 = 等同于禁用

# Boss 池刷新规则
- Mod 更新或 Boss 池重新初始化后，筛选器会重新构建列表。
- 系统会自动剔除 Mode E / Mode F 临时生成的杂兵预设，避免筛选器被杂兵污染。
- 龙裔、龙王以及少数特殊 Boss 底层预设会被保留，不会因为这类清理而从筛选器里消失。

# 数据保存
- 筛选器的设置会保存到配置文件中，下次启动游戏时自动加载。
