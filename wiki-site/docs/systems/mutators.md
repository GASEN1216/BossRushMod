# 变异词条系统

# 这是啥？

每局开始前，系统会从词条池随机抽取 **1~3 个变异词条**，立刻生效并贯穿整局。词条可能给敌人增益、给玩家增益，也可能改变环境规则——结果随机，每局不同。

> **末日丧尸模式例外**：丧尸模式有自己独立的局内奖励增益系统，不走这套机制。

# 适用模式

| 模式 | 是否抽词条 |
|------|---------|
| 标准 BossRush | ✅ |
| 无间炼狱 | ✅ |
| 白手起家 | ✅ |
| 划地为营 | ✅ |
| 血猎追击 | ✅ |
| 末日丧尸模式 | ❌（独立系统） |

# 如何查看当前词条？

- 局开始后，屏幕**左侧边缘**保留当前词条简要提示（约屏幕高度 25% 处）
- 鼠标移到左侧词条提示上时显示完整效果说明，移开后自动隐藏

# 配置

- **开关**：`enableMutators`（默认 `true`）
- **数量**：`mutatorCount`，范围 1~3，默认 **2** 个/局
- 可通过 ModConfig UI 或配置文件修改

# 常见问题

**Q: 好词条和坏词条概率一样？**  
随机从池里抽取，不区分对玩家有利与否。同一局可能同时抽到「巨灵兵」和「玻璃大炮」。

**Q: 能不能只抽玩家增益词条？**  
不行，词条是无差别抽取的，不能指定类别。

**Q: 词条会在局结束后清除吗？**  
会，所有词条在任意形式的局结束（通关/死亡/主动撤离）后自动清除，不会污染下一局。

**Q: 天降殉爆真的会伤到我？**  
会。敌人死亡时触发的爆炸会对玩家造成伤害，建议保持距离击杀，避免被连环引爆。有内置防递归保护，不会造成无限连锁。
