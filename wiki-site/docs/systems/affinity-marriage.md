# 好感度与婚姻

## 好感度系统

叮当和羽织拥有完整的好感度系统，共 10 个等级。提升好感度可以解锁新服务、折扣和剧情。

### 等级与所需点数

| 等级 | 累计所需点数 | 本级所需 |
|------|------------|---------|
| Lv.1 | 0（初始） | — |
| Lv.2 | 50 | 50 |
| Lv.3 | 150 | 100 |
| Lv.4 | 300 | 150 |
| Lv.5 | 500 | 200 |
| Lv.6 | 750 | 250 |
| Lv.7 | 1,050 | 300 |
| Lv.8 | 1,400 | 350 |
| Lv.9 | 1,800 | 400 |
| Lv.10 | 2,300 | 500 |

### 获取好感度的方式

| 方式 | 叮当 | 羽织 | 频率 |
|------|------|------|------|
| 每日聊天 | +40 | +30 | 每天 1 次（好感度收益） |
| 送喜欢的礼物 | +80 | +80 | 每天 1 次 |
| 送普通礼物 | +20 | +20 | 每天 1 次 |
| 送讨厌的礼物 | -40 | -40（砖石为 -60） | 每天 1 次 |
| 送钻石戒指 | +500 | +500 | — |
| 送生日蛋糕 | +150 | +150 | — |

### 好感度衰减

如果某一天你没有与 NPC 互动（聊天或送礼），第二天好感度会 -15。保持每日互动可以避免衰减。

### 各等级解锁内容

详见 [叮当](../npcs/goblin.md) 和 [羽织](../npcs/nurse.md) 的各自页面。

## 婚姻系统

### 求婚条件

1. 与目标 NPC 的好感度达到 Lv.10
2. 建造婚礼教堂（需要先放置建筑）
3. 向 NPC 赠送钻石戒指

### 婚礼流程

1. 送出钻石戒指后，NPC 接受求婚
2. 播放全屏婚礼过场动画（如果动画文件不存在则显示对话序列）
3. NPC 头顶出现爱心气泡和纪念日期
4. NPC 搬迁到婚礼教堂

### 婚后变化

| 变化 | 说明 |
|------|------|
| NPC 位置 | 搬到婚礼教堂 |
| 对话内容 | 切换为婚后专属对话 |
| 叮当特殊福利 | 每日聊天赠送冷淬液 |
| 羽织特殊福利 | 每日聊天赠送安神滴剂 |
| 重播婚礼 | 可在教堂重播婚礼过场 |

### 婚姻规则

| 规则 | 说明 |
|------|------|
| 一夫一妻 | 同一时间只能与一位 NPC 结婚 |
| 重复求婚 | 已婚状态下向配偶再次送戒指会被拒绝，戒指不消耗 |
| 出轨惩罚 | 已婚状态下向其他 NPC 送戒指：戒指被消耗，好感度下降 120 点，配偶记录出轨事件并在下次聊天时责备（多次出轨每次额外扣 40 点） |

### 离婚

- 在婚礼教堂与配偶互动，选择"离婚"
- 离婚后好感度归零
- NPC 搬回原来的位置
