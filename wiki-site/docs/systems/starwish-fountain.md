# 布满了灰尘的星愿许愿台

# 这是什么

- 布满了灰尘的星愿许愿台是基地中的可建造互动建筑。
- 它允许你把想法、建议或希望看到的内容直接写成心愿并提交。
- 当前实现通过飞书多维表格接收这些心愿，不需要额外服务器。

# 如何使用

- 在基地建造界面中建造布满了灰尘的星愿许愿台。
- 靠近后会看到交互选项 `许愿`。
- 打开后会进入一个原版风格的运行时 View 面板，而不是旧版 IMGUI 窗口。

# 当前 UI 规则

- 标题是 `写下你的心愿`。
- 输入区是固定高度的多行文本框。
- 输入过长时会使用输入区内部滚动条，不会继续把整个窗口撑高。
- 右侧固定提示文案为：`请不要输入无效/垃圾内容哦~`
- 匿名勾选框当前默认关闭。
- 字数范围为 `20 ~ 10000`。

# 提交后的反馈

- 发送中会显示 `正在将心愿送往星空…`
- 成功后会显示原版大横幅：
  - `愿这点星光，照亮你心中的乌托邦~`
- 面板会在成功后约 1 秒自动关闭。
- 当前冷却是全局 `30 秒`，不是每个许愿台分别计时。

# 许愿抽奖奖励

- 每次成功提交心愿后，系统都会额外检查一次 `许愿抽奖` 是否可用。
- 抽奖奖励冷却是全局 `4 小时`，不是每个许愿台分别计时。
- 如果奖励冷却已经结束，提交成功后会进入 `星愿抽奖` 动画，并在结束后发放 1 件奖励物品。
- 如果奖励仍在冷却中，你的心愿依然会正常提交，但只会显示剩余抽奖冷却时间，不会重复发奖。
- 心愿内容会影响奖励倾向：
  - 写到某些物品类型或关键词时，会更容易抽到对应方向的奖励。
  - 当前会偏向武器、近战、护甲、头盔、图腾、礼物、治疗、营旗、召唤道具、工事、旅行类物品等方向。
- 抽奖时可以按 `Esc` 直接跳过动画，奖励仍会正常结算。

# 匿名与隐私

- 勾选匿名后，会以 `匿名玩家` 的名字提交。
- 不勾选匿名时，会尝试使用你的 Steam 昵称。
- 如果 Steam 昵称获取失败，也会自动回退到匿名。
- 不会提交 Steam ID 之类的唯一身份标识。

# 内容限制

- 纯空白、无意义刷屏内容、重复内容会被拦截。
- 包含链接、联系方式、引流内容的文本会被拦截。
- 包含辱骂、敏感词或广告内容的文本会被拦截。

::: tip
如果提交按钮不可用，先检查字数是否达到 20 字，或是否还在 30 秒全局冷却中。提交成功后如果没有触发抽奖，再看是否还在 4 小时奖励冷却中。
:::
