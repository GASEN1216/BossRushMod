# 龙王套装

## 概述

龙王套装是龙裔套装的高阶版本，由焚天龙皇掉落。保留了火焰转治疗的核心被动，同时大幅强化了冲刺能力并移除了部分负面属性。

## 套装部件

### 龙王之冕（头盔）

| 属性 | 数值 |
|------|------|
| 护甲 | 7 |
| 耐久 | 200 |
| 物理抗性 | -15% |
| 风暴抗性 | +1 |
| 寒冷抗性 | +1 |
| 暴击伤害 | +15% |
| 火焰抗性 | -20% |
| 电击抗性 | -20% |

相比赤龙首：暴击伤害更高（+15% vs +10%），移除了毒素易伤和视野缩小。

掉落来源：[焚天龙皇](../bosses/dragon-king.md)（15% 掉率）

### 龙王鳞铠（护甲）

| 属性 | 数值 |
|------|------|
| 护甲 | 7 |
| 耐久 | 200 |
| 物理抗性 | -25% |
| 风暴抗性 | +1 |
| 寒冷抗性 | +1 |
| 火焰抗性 | -25% |
| 电击抗性 | -25% |

相比焰鳞甲：移除了毒素易伤。

掉落来源：[焚天龙皇](../bosses/dragon-king.md)（15% 掉率）

## 套装效果（两件全穿）

| 效果 | 说明 |
|------|------|
| 火焰转治疗 | 受到的火焰伤害不再造成伤害，反而回复 80% 等量的生命值 |
| 龙眼光效 | 头部出现红色呼吸灯光效 |
| 龙王冲刺 | 首段冲刺 6 米，冷却 0.5 秒 |
| 二段冲刺 | 首段冲刺后 0.15 秒内按方向键可追加 3 米二段冲刺 |
| 熔浆轨迹 | 冲刺路径留下熔浆地带，对敌人造成持续伤害（每 0.5 秒 5 点，持续 3 秒） |
| 金橙残影 | 冲刺时留下金橙色残影（区别于龙裔套装的橙红色） |

## 与龙裔套装对比

| 对比项 | 龙裔套装 | 龙王套装 |
|--------|---------|---------|
| 冲刺距离 | 3 米 | 6 米 + 3 米二段 |
| 冲刺冷却 | 1.5 秒 | 0.5 秒 |
| 熔浆轨迹 | 无 | 有（对敌人造成伤害） |
| 毒素易伤 | 有 | 无 |
| 视野缩小 | 有（赤龙首 -20%） | 无 |
| 暴击伤害 | +10% | +15% |

龙王套装是全面升级，获取后建议直接替换龙裔套装。
