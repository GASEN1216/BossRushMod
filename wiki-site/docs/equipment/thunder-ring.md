# 这是啥？

一件**图腾槽蓄力装备**——被打越多，下一击越猛。每次受击积累 1 层雷电能量，满 5 层后，你下一次近战命中会附带一次 40 点雷属性爆发伤害。适合"以肉换伤"的激进玩法。

- **类型**：图腾 / **品质**：5
- **获取**：暂无常规掉落来源（当前版本）
- **死亡掉落**：不会因死亡掉落

# 雷霆蓄力（被动）

**蓄力规则：**
- 每次受击积累 **1 层雷电**（最多 5 层）
- 连续受击有 **0.3 秒**冷却，防止瞬间满层
- 最后一次受击后 **8 秒内**未再受击，全部层数消散

**释放规则：**
- 当层数达到 **5 层**时，你的下一次近战攻击命中会附加 **40 点雷属性伤害**，并清空所有层数
- 释放后可以重新蓄力

# 实战体验

- 本质上是"以受伤换伤害加成"——适合主动冲上去打的激进玩法
- 面对单个慢速 Boss 时稳定蓄层，爆发可靠
- 低伤害/空弹药阶段（敌人少）几乎无法蓄层
- 图腾槽位，不占武器位，可以叠加任意近战武器效果
- 建议配合高暴击/高单次伤害武器（如焚皇断界戟）一起使用，等爆发触发时用强力攻击输出

::: tip
不要刻意"省"第5层——雷电伤害固定40点，早触发早蓄力，总输出反而更高。
:::
