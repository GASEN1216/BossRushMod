# 龙息

## 概述

龙息是龙裔遗族掉落的专属枪械，命中敌人时有概率施加龙焰灼烧效果，造成持续的真实伤害。它是玩家能获取的第一把自定义武器。

## 基础信息

| 属性 | 说明 |
|------|------|
| 类型 | 枪械 |
| 掉落来源 | [龙裔遗族](../bosses/dragon-descendant.md)（10% 掉率） |
| 死亡掉落 | 正常掉落 |

## 特殊效果

| 效果 | 说明 |
|------|------|
| 龙焰灼烧 | 每次命中 50% 概率施加灼烧 Debuff |
| 灼烧伤害 | 每层每跳 1 点真实伤害（无视护甲） |
| 可叠加 | 多次触发可叠加多层灼烧，持续累积伤害 |

## 使用建议

- 龙息的核心价值在于持续的灼烧真伤，适合对付高护甲目标
- 50% 的触发率意味着高射速下可以快速叠加多层灼烧
- 作为龙裔遗族的掉落物，它是你接触自定义装备体系的起点
- 配合 [龙裔套装](dragon-set.md) 使用，形成完整的龙裔装备组合
