# 龙息

# 概述
- 龙裔遗族掉落的专属枪械，你能获取的第一把自定义武器
- 命中敌人时有概率施加龙焰灼烧效果，造成持续的真实伤害
- 枪口带有火焰烟雾和火星特效

# 基础信息
- 类型：枪械
- 掉落来源：龙裔遗族（10% 掉率）
- 死亡掉落：正常掉落

# 武器数据
- 基础伤害：23
- 射速：13
- 弹匣容量：20
- 装填时间：3 秒
- 弹速：122
- 射程：28
- 暴击率：25%
- 暴击伤害：1.5 倍
- 持枪移速：82%
- 瞄准移速：45%
- 散布（腰射）：27
- 散布（瞄准）：8
- 元素：火焰

# 龙焰灼烧效果
- 触发概率：每次命中 50%
- 灼烧伤害：每层每秒造成（目标最大生命值的 0.1% + 1）点真实火焰伤害
- 最大叠加层数：10 层
- 持续时间：10 秒
- 对玩家伤害上限：每跳 2 点（基础伤害封顶）
- 火焰抗性可减免灼烧伤害

# 使用建议
- 核心价值在于百分比最大生命值灼烧，对高血量 Boss 尤其有效
- 射速 13 配合 50% 触发率，可以快速叠满 10 层灼烧
- 10 层满叠加时，每秒造成（目标最大生命值的 1% + 10）点真伤，非常可观
- 配合龙裔套装使用，形成完整的龙裔装备组合

# 相关成就
- 龙裔珍藏 — 收集龙裔遗族所有专属掉落（赤龙首、焰鳞甲、龙息）

::: tip
龙息是你接触自定义装备体系的起点，别小看它的灼烧叠加——对高血量目标的持续输出远超面板伤害。
:::
