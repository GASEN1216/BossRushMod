# 这是啥？

**雷系双件套**，由头盔「雷神之角」与护甲「雷霆战甲」组成。单件效果有限，同时装备两件则激活「**雷霆之怒**」——电系减伤加上近战反击爆炸，把攻击你的敌人炸成筛子。

# 套装组合

| 物品 | 槽位 | 品质 | 加成 | ID |
|------|------|------|------|-----|
| **雷神之角** | 头盔 | 6 | 头甲 +5 | 500055 |
| **雷霆战甲** | 护甲 | 6 | 体甲 +5 | 500056 |

- 耐久度：999（近似永久）
- **获取**：暂无常规掉落来源（当前版本）
- **死亡掉落**：不会因死亡掉落

# 2 件套效果：雷霆之怒

同时装备雷神之角 + 雷霆战甲时激活，屏幕上方会出现提示。

**被动：**
- 电属性伤害减免 **+50%**

**触发（反击爆炸）：**
- 被距离 **6 米以内**的攻击者命中时，有 **25%** 概率在自身位置释放**电击 AOE**
  - 半径 **4 米** / 伤害 **25 点电属性**
- 每 **3 秒**最多触发一次
- 对自身无伤害（`canHurtSelf=false` 已设定，不会误伤自己）

# 与霜冠套装的区别

| | 霜冠套装（寒冰之护） | 雷神套装（雷霆之怒） |
|--|--|--|
| 减伤类型 | 冰 | 电 |
| 触发范围 | 5 米 | 6 米 |
| 触发概率 | 30% | 25% |
| 效果 | 冻结攻击者 | AOE 爆炸伤害 |
| 冷却 | 5 秒 | 3 秒 |

# 实战体验

- 电系减伤在末日丧尸模式里对骚扰者（Harasser）丧尸有用
- 3 秒冷却比霜冠套装短，在混战中爆炸触发更频繁
- AOE 4 米范围——适合群敌密集的场景，一炸一片
- 反击伤害（25 点）不高，但叠加次数多时合计可观

::: tip
雷神套装在多敌密集的无间炼狱/末日丧尸波次中表现更好；霜冠套装更适合 1v1 控制场景。按需选择。
:::
