# 焚天龙铳

# 概述
- 焚天龙皇在战斗中使用的专属枪械
- 玩家无法通过掉落或其他方式获取，仅 Boss 自身装备

::: warning
这把武器被列入掉落黑名单，不会出现在任何战利品箱中。
:::

# 基础信息
- 类型：枪械
- 持有者：焚天龙皇（Boss 专属）
- 玩家可获取：否

# 武器数据
- 基础伤害：26
- 射速：9.2
- 弹匣容量：15
- 装填时间：3.35 秒
- 弹速：108
- 射程：24
- 暴击率：28%
- 暴击伤害：1.6 倍
- 元素：火焰
- 持枪移速：85%

# 弹药系统
- 焚天龙铳支持 15 种弹药配置
- 每种弹药会动态改变武器的射速、伤害、弹匣容量、装填时间和射程等参数
- 这使得焚天龙皇在战斗中的射击表现具有一定的变化性

# 为什么要了解它
- 了解焚天龙铳的数据有助于你在面对焚天龙皇时评估其射击伤害
- 如果你想获取玩家可用的龙系枪械，可以挑战龙裔遗族获取龙息
