# 焚天龙铳

# 这是啥？

焚天龙皇的私人武器——兼容 15 种弹药，玩家可通过弹药类型 UI 选择并装填；每一种都会被龙铳改写成不同的龙焰弹幕。

- **类型**：枪械 / **元素**：火焰
- **掉落**：焚天龙皇（**1%** 掉率，极其稀有）
- **玩家可用**：是

# 面板数据

- 基础伤害：26 / 基础射速：9.2 / 基础弹匣：15 / 基础装填：3.35 秒
- 基础弹速：108 / 基础射程：24 / 暴击率：28% / 暴击伤害：1.6 倍
- 持枪移速：85%

::: tip
换装不同弹种后，伤害、射速、弹匣、装填、射程与弹道会随 Profile 改写；上方为未装填特定弹种时的基础面板。
:::

# 15 种可选弹药——每一发都不一样

背包里有兼容弹药时，焚天龙铳会在弹药类型列表中显示它们；选择并换弹后即可使用对应弹幕。

# 暴力输出型
- **狙击弹** — 4 穿透，射速极慢（×0.12），伤害极高（×4），射程 36；适合精确点杀。
- **重弹** — 2 穿透，后续穿透命中伤害衰减，伤害 ×1.4，射程 28。
- **火箭弹** — 低弧线追踪火箭，在约半程空爆，径向分裂为 **6** 枚子火箭；弹匣仅 5 发。
- **冰刃弹** — 冰元素，3 穿透，伤害 ×2，射速减半；命中或终点产生 1.25 米冰爆并施加冰冻效果。

# 弹幕覆盖型
- **霰弹** — 12 发、20 度龙息喷扫，伤害 ×1.65；命中或射程尽头最多生成 2 片灼烧地带。
- **突击弹** — 3 发散射，弹道可反弹 1 次。
- **冲锋弹** — 2 发螺旋弹道，射速 ×1.6，弹匣 30 发。
- **糖果弹** — 6 发、180 度随机散射，弹道可反弹；每颗糖果保留独立伤害。

# 花式弹道型
- **马格南弹** — 3 穿透，弹道会回旋，伤害 ×1.9。
- **箭矢弹** — 轻微下坠，命中后黏附并在短延迟后引爆，伤害 ×1.8。
- **能量弹** — 电击元素、追踪弹道，命中或结束后分裂 3 发追踪子弹。
- **纳米弹** — 毒元素高速毒针，具有 3 穿透与 3 次反弹；不再分裂。
- **烟花弹** — 水平螺旋弹道，飞行至约七成射程时空爆，分裂为 **12** 枚烟花子弹。首个命中目标不会被分裂弹再次集火，子弹按错峰调度保持爆裂视觉并降低命中卡顿。

# 整活元素型
- **雪球弹** — 冰元素，低基础伤害、超慢速直线滚动 5 秒并持续变大，伤害随体积提高；主雪球命中或寿命结束后分裂 4 个小雪球。小雪球有短暂生成保护、滚动 2 秒并继续成长；只有主雪球会留下 1 秒冰区。
- **便便弹** — 毒元素，高抛物线，伤害 ×3.2，落地生成持续 4 秒的毒池，射程 60。

# 龙焰印记联动

- 龙铳命中会叠加龙焰印记，单个目标最多 **10 层**。
- 切换 **焚皇断界戟** 后，用砸落命中带印记目标可引爆全部层数；每层造成 30 点爆燃伤害。

# 使用建议

- 火箭、烟花与霰弹适合清理密集敌群；狙击、重弹与马格南更适合处理单个高威胁目标。
- 雪球、毒池和灼烧地带需要预判落点；能量和纳米适合应对走位目标。
- 烟花弹的性能优化不削弱特效或伤害，但近距离爆裂时仍建议避免把所有子弹都压在狭窄地形中。
