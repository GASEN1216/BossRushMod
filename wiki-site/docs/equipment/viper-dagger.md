# 这是啥？

一把**毒属性短刃**，靠叠毒反复削血——单次伤害不高，但连续命中会叠满 5 层毒素，在第5层触发一次爆发追加伤害。贴脸快速连斩的风格。

- **类型**：近战武器 / **品质**：5 / **元素**：毒
- **获取**：暂无常规掉落来源（当前版本）
- **死亡掉落**：不会因死亡掉落

# 面板数据

- 伤害：22 / 攻速：2.1 / 范围：1.4 米
- 暴击率：8% / 暴击伤害：1.5 倍 / 护甲穿透：2
- 格挡子弹：0.2 / 体力消耗：4/击 / 出伤时间：0.06 秒
- 持械移速：112% / 出血概率：0%

# 蛇毒注射（被动）

每次近战命中对目标附加 **1 层毒素**（最多叠 5 层）：

- 同一目标超过 **6 秒**未被命中，毒层全部消散
- 叠满 **5 层**时，自动触发**蛇毒爆发**：造成 **35 点毒属性爆发伤害**，同时清除全部层数
- 爆发后可重新开始叠毒

# 实战体验

- 攻速快（2.1），配合低体力消耗（4/击）可以快速叠毒
- 爆发伤害不走暴击和武器面板，始终稳定造成 35 点
- 范围只有 1.4 米——必须贴近才能打，被包围时风险高
- 暴击面板低，前期单次输出偏弱；依赖叠毒补偿总伤害

::: tip
对速度快、难以贴近的敌人（如猎手 Boss）表现平平；对站桩输出型 Boss 效果更好，能稳定打满 5 层。
:::
