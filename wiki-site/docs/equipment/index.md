# 装备总览

BossRush Mod 添加了一系列自定义装备，主要来自三条掉落线：龙裔遗族、焚天龙皇、幽灵女巫。
此外，原版 `Cname_Boss_Blue` 现在也有概率额外掉落一把自定义近战武器：霜之哀伤。

# 装备一览
- 龙裔套装 — 头盔 + 护甲（龙裔遗族掉落）
- 龙王套装 — 头盔 + 护甲（焚天龙皇掉落）
- 腾云驾雾图腾 — 图腾（焚天龙皇掉落）
- 逆鳞 — 图腾（焚天龙皇掉落）
- 焚皇断界戟 — 近战武器（焚天龙皇掉落）
- 噬魂挽歌 — 幽灵属性近战武器（幽灵女巫 50% 额外掉落）
- 龙息 — 枪械（龙裔遗族掉落）
- 焚天龙铳 — 枪械（焚天龙皇掉落，5%）
- 霜之哀伤 — 冰属性近战武器（`Cname_Boss_Blue` 50% 额外掉落）

# 套装效果

两套龙系套装都提供"火焰伤害转治疗"的核心被动，区别在于冲刺能力的强度：

- 龙裔套装 — 冲刺距离 3 米，冷却 1.5 秒，双击方向键触发
- 龙王套装 — 冲刺距离 6 米（可追加 3 米二段冲刺），冷却 0.5 秒，冲刺路径留下熔浆灼烧

# 图腾对比
- 腾云驾雾 — 飞行/滑翔机动，持续使用，消耗体力
- 逆鳞 — 濒死保命反扑，一次性触发后碎裂

# 装备获取建议
- 先挑战龙裔遗族，获取龙裔套装和龙息作为过渡
- 熟悉龙裔遗族后挑战焚天龙皇，追求龙王套装和专属武器
- 想走高机动与控场路线时，可以继续挑战幽灵女巫获取噬魂挽歌
- 图腾根据玩法风格选择：追求机动选腾云驾雾，追求保命选逆鳞
- 如果你会刷原版 Blue Boss，可以顺手蹲霜之哀伤；它的冰属性和右键亡灵召唤都很适合 BossRush
