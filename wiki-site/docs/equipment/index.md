# 装备总览

BossRush 里的原创装备一共 9 件，主要从三个 Boss 身上薅：龙裔遗族、焚天龙皇、幽灵女巫。另外原版的 `Cname_Boss_Blue` 也有概率掉一把冰系近战——霜之哀伤。

# 全家福

- **龙裔套装**（头盔+护甲）— 龙裔遗族掉落，入门级自定义装备
- **龙王套装**（头盔+护甲）— 焚天龙皇掉落，龙裔套装的豪华升级版
- **腾云驾雾图腾** — 焚天龙皇掉落，装上就能飞
- **逆鳞** — 焚天龙皇掉落，濒死反杀的一次性保命神器
- **焚皇断界戟** — 焚天龙皇掉落，三连击+地面炸裂的近战大杀器
- **噬魂挽歌** — 幽灵女巫 50% 额外掉落，幽灵属性镰刀
- **龙息** — 龙裔遗族掉落，能叠灼烧的火焰枪
- **焚天龙铳** — 焚天龙皇掉落（5%），15 种弹药随机切换的魔性枪
- **霜之哀伤** — `Cname_Boss_Blue` 50% 额外掉落，冰属性大剑，右键召唤亡灵

# 两套龙系套装怎么选？

两套都有"**火焰伤害转治疗**"的核心被动（打龙系 Boss 简直是挂），区别在冲刺：

- **龙裔套装** — 冲刺 3 米，冷却 1.5 秒，双击方向键触发。够用
- **龙王套装** — 冲刺 6 米（可追加 3 米二段冲刺），冷却 0.5 秒，路径留下熔浆灼烧。**全方位碾压**

简单来说：龙裔套装是过渡装，龙王套装是毕业装。

# 两个图腾怎么选？

- **腾云驾雾** — 飞行/滑翔，持续消耗体力。适合追求高机动
- **逆鳞** — 濒死时触发，回血+反击，用完就碎。适合保命党

# 装备获取路线

- **第一站**：先打龙裔遗族，拿龙裔套装 + 龙息，够用一阵
- **第二站**：熟悉之后挑战焚天龙皇，追龙王套装和毕业武器
- **支线**：想玩近战控场路线的话，挑战幽灵女巫拿噬魂挽歌
- **隐藏线**：刷原版 Blue Boss 顺手蹲霜之哀伤，冰属性+亡灵召唤，在 BossRush 里非常好使
