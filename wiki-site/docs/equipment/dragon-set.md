# 龙裔套装

## 概述

龙裔套装由龙裔遗族掉落的两件装备组成，是玩家接触到的第一套自定义套装。集齐两件后激活套装效果，提供火焰免疫和冲刺能力。

## 套装部件

### 赤龙首（头盔）

| 属性 | 数值 |
|------|------|
| 护甲 | 7 |
| 耐久 | 200 |
| 物理抗性 | -10% |
| 风暴抗性 | +1 |
| 寒冷抗性 | +1 |
| 暴击伤害 | +10% |
| 火焰抗性 | -15% |
| 电击抗性 | -15% |
| 毒素易伤 | +30% |
| 视野角度 | -20% |

掉落来源：[龙裔遗族](../bosses/dragon-descendant.md)（30% 掉率）

### 焰鳞甲（护甲）

| 属性 | 数值 |
|------|------|
| 护甲 | 7 |

| 物理抗性 | -20% |
| 风暴抗性 | +1 |
| 寒冷抗性 | +1 |

| 电击抗性 | -20% |

掉落来源：[龙裔遗族](../bosses/dragon-descendant.md)（60% 掉率）

## 套装效果（两件全穿）

| 效果 | 说明 |
|------|------|
| 火焰转治疗 | 受到的火焰伤害不再造成伤害，反而回复 80% 等量的生命值 |
| 龙眼光效 | 头部出现红色呼吸灯光效 |
| 冲刺 | 双击方向键触发冲刺，距离 3 米，冷却 1.5 秒，留下橙红色残影 |

冲刺功能可在配置中关闭（`enableDragonDash`）。

## 使用建议

- 龙裔套装的负面属性不少（毒素易伤、视野缩小），但火焰免疫和冲刺带来的生存能力提升远大于代价
- 火焰转治疗在面对龙裔遗族时尤其强力——Boss 的龙息反而会给你回血
- 如果后续获得了 [龙王套装](dragon-king-set.md)，建议直接升级替换
