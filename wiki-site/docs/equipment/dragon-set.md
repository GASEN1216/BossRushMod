# 龙裔套装

# 概述
- 龙裔套装是你接触到的第一套自定义套装，由龙裔遗族掉落
- 集齐两件激活套装效果：火焰免疫 + 冲刺能力
- 负面属性不少，但带来的生存能力提升远大于代价

# 套装部件

# 赤龙首（头盔）
- 护甲：7
- 耐久：200
- 物理抗性：-10%
- 风暴抗性：+1
- 寒冷抗性：+1
- 暴击伤害：+10%
- 火焰抗性：-15%
- 电击抗性：-15%
- 毒素易伤：+30%
- 视野角度：-20%
- 掉落来源：龙裔遗族（30% 掉率）

# 焰鳞甲（护甲）
- 护甲：7
- 耐久：200
- 物理抗性：-20%
- 风暴抗性：+1
- 寒冷抗性：+1
- 火焰抗性：-20%
- 电击抗性：-20%
- 毒素易伤：+40%
- 掉落来源：龙裔遗族（60% 掉率）

# 套装效果（两件全穿）
- 火焰转治疗：受到的火焰伤害不再造成伤害，反而回复 80% 等量的生命值
- 龙眼光效：头部出现红色呼吸灯光效
- 冲刺：双击方向键触发，距离 3 米，冷却 1.5 秒，留下橙红色残影

::: tip
冲刺功能可在配置中关闭（enableDragonDash）。
:::

# 使用建议
- 火焰转治疗在面对龙裔遗族时尤其强力——Boss 的龙息反而会给你回血
- 毒素易伤和视野缩小是主要代价，打毒系敌人时要注意
- 如果后续拿到了龙王套装，建议直接升级替换
- 先把龙裔线玩明白，再追龙王线，体验会更完整
