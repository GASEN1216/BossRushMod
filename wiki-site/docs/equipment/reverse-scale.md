# 逆鳞

# 这是啥？

**龙之逆鳞，触之必怒。** 一次性保命图腾——你要死的那一刻它替你挡一刀，回血 + 向四周发射棱彩弹反击，然后碎掉。简单粗暴，救命用的。

- **类型**：图腾 / **品质**：6（传说）
- **掉落**：焚天龙皇（39% 掉率，龙皇掉落里最容易出的）
- **死亡掉落**：不会因死亡掉落

# 怎么触发的？

- 血量降到 **1 或以下**时自动触发
- 回复**最大生命值的 50%**
- 向 8 个方向发射棱彩弹，自动追踪 15 米内最近的敌人
- 每颗 15 点伤害，追踪 2.5 秒后转直线，总共存在 5 秒
- 触发后获得 **0.5 秒短暂无敌**，随后图腾碎裂并**从装备槽里消失**

::: warning
触发时屏幕会显示："逆鳞碎了..."——意思是你刚才差点死了，而且这个保命机会用掉了。
:::

# 什么时候装？

- **首次挑战焚天龙皇**——不确定能不能打过的时候，带一个心里踏实
- **无间炼狱高波次**——后期一个失误就可能暴毙，逆鳞给你一次翻盘的机会
- **血猎追击**——持续失血 + Boss 追杀，多条命总比一条命好
- 稳定期可以换腾云驾雾图腾，**高风险期换逆鳞**，灵活切换
