# 逆鳞

# 概述
- 龙之逆鳞，触之必怒
- 一次性保命图腾——濒死时自动触发，回复生命并向四周发射棱彩弹反击，随后图腾碎裂

# 基础信息
- 类型：图腾
- 品质：6（传说）
- 掉落来源：焚天龙皇（35% 掉率）
- 死亡掉落：不会因死亡掉落

# 触发机制
- 触发条件：生命值降至 1 或以下
- 回复量：最大生命值的 50%
- 反击：向 8 个方向发射棱彩弹，自动追踪 15 米内最近的敌人
- 棱彩弹伤害：15 点/颗
- 棱彩弹追踪：追踪 2.5 秒后转为直线飞行，总存在时间 5 秒
- 触发后：图腾碎裂并从装备槽中移除

::: warning
触发时会显示提示文字："逆鳞碎了..."
:::

# 使用建议
- 逆鳞是纯粹的保险机制，适合高风险场景（首次挑战焚天龙皇、无间炼狱高波次、血猎追击等）
- 一次性使用，触发后需要重新获取
- 棱彩弹的追踪伤害在近距离多敌人场景下效果最好
- 可以和腾云驾雾图腾轮换使用——稳定期用飞行图腾，高风险期换逆鳞
