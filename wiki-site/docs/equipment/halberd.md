# 焚皇断界戟

# 这是啥？

焚天龙皇掉落的专属近战武器——**Mod 里机制最丰富的一把**。三段连击 + 龙焰印记 + 跳劈引爆 + 火柱炸裂，一套组合拳打下来，场面堪比龙皇本龙。

- **类型**：近战武器 / **品质**：7（传说）/ **元素**：火焰
- **掉落**：焚天龙皇（15% 掉率）
- **死亡掉落**：不会因死亡掉落

# 面板数据

- 基础伤害：55 / 攻速：2.2 / 范围：2.5 米
- 暴击率：10% / 暴击伤害：2.0 倍 / 护甲穿透：6
- 出血概率：15% / 格挡子弹：1 / 体力消耗：10/击
- 持械移速：110% / 出伤时间：0.12 秒 / 宝石槽位：2

# 三段连击（左键）——一段比一段猛

连续左键触发三段连击，0.75 秒内连上就不断，超时重置：

- **第 1 段「横扫炎刃」** — 2.8 米 150° 扇形，55 伤害
- **第 2 段「龙焰挑天」** — 2.2 米 90° 扇形，65 伤害，**把敌人挑飞 1.5 米**
- **第 3 段「焚界重劈」** — 3.3 米 120° 扇形，85 伤害，**把敌人拉向自己 1 米**

每段攻击都附带：+15 火焰附加伤害 / 2 秒灼烧 / **叠 1 层龙焰印记**

# 龙焰印记——叠满了就炸

- 最多 5 层/敌人，持续 6 秒（命中刷新计时）
- 引爆方式：龙皇裂地落地时自动引爆范围内所有印记
- **每层 30 点引爆伤害**（5 层满印记 = 150 点额外伤害，一套连招带走）

# 龙皇裂地（右键）——跳起来砸下去

- 冷却 5 秒 / 体力消耗 20
- **按住右键**显示跳跃弧线预览，**松开**执行跳劈
- 橙色弧线 = 可以落，红色 + 禁止图标 = 被挡了

# 落地炸什么？

- **冲击伤害** — 80 点（3.5 米范围）
- **火柱** — 落地点周围 6 根（半径 1.8 米），每根 45 伤害，1.2 米范围，持续 2 秒
- **印记引爆** — 自动引爆范围内所有敌人身上的龙焰印记

# 最佳战斗循环

- 三段连击叠满 5 层龙焰印记
- 右键跳劈引爆印记 + 铺火柱
- 冷却期间继续连击补层
- **如此循环，伤害爆炸**

# 实战心得

- 第 2 段挑飞 + 第 3 段拉拽 = **控制敌人位置**，配合跳劈范围伤害绝了
- 跳劈有预览弧线，别盲跳——**瞄准敌人密集处**效果最大化
- 配合龙王套装冲刺可以快速贴脸开连击

::: tip
这把武器的上限极高，但需要你理解连击节奏和跳劈时机。练好了就是近战之王。
:::
