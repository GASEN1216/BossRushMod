# 焚皇断界戟

# 概述
- 焚天龙皇掉落的专属近战武器，Mod 中机制最丰富的近战武器
- 拥有三段连击系统和龙皇裂地跳劈技能

# 基础信息
- 类型：近战武器
- 品质：7（传说）
- 掉落来源：焚天龙皇（15% 掉率）
- 死亡掉落：不会因死亡掉落

# 三段连击（左键）
连续点击左键触发三段连击，每段有不同的攻击范围和效果。连击窗口为 0.75 秒，超时则重置。

- 第 1 段「横扫炎刃」：范围 2.8 米，角度 150°，伤害 55
- 第 2 段「龙焰挑天」：范围 2.2 米，角度 90°，伤害 65，将敌人挑飞 1.5 米
- 第 3 段「焚界重劈」：范围 3.3 米，角度 120°，伤害 85，将敌人拉向自己 1.0 米

所有攻击附带：
- +15 火焰附加伤害
- 2 秒灼烧 Debuff
- 叠加 1 层龙焰印记（每次命中）

# 龙焰印记
- 最大层数：5 层/敌人
- 持续时间：6 秒（每次叠加刷新）
- 引爆方式：龙皇裂地落地时自动引爆范围内所有印记
- 引爆伤害：每层 30 点（5 层满印记 = 150 点）

# 龙皇裂地（右键）
- 冷却时间：5 秒
- 体力消耗：20
- 操作方式：按住右键显示跳跃预览弧线，松开执行
- 预览指示：橙色 = 可落地，红色 + 禁止图标 = 被阻挡

# 落地效果
- 落地冲击伤害：80 点（3.5 米范围）
- 火柱：落地点周围环形生成 6 根火柱（半径 1.8 米），每根 45 伤害，1.2 米范围，持续 2 秒
- 印记引爆：自动引爆范围内所有敌人的龙焰印记

# 推荐战斗循环
1. 三段连击叠满 5 层龙焰印记
2. 右键跳劈引爆印记 + 火柱覆盖
3. 等待冷却期间继续连击叠印记
4. 循环

# 使用建议
- 连击的第 2 段挑飞和第 3 段拉拽可以控制敌人位置，配合跳劈的范围伤害效果极佳
- 跳劈预览弧线可以帮你精确选择落点，不要盲目释放
- 火柱以落点为中心环形分布，跳劈到敌人密集处可以最大化覆盖
- 配合龙王套装的冲刺可以快速接近目标后连击

::: tip
这把武器的上限极高，但需要你理解连击节奏和跳劈时机。
:::
