# Boss 总览

原版的 Boss 不够打？BossRush Mod 额外加了 3 个全原创 Boss——每个都有独立技能、阶段转换和专属掉落。它们不是换个皮肤的杂兵，是真正需要你研究打法的硬茬。

# 三大原创 Boss

- **龙裔遗族**（500 血）— 入门级自定义 Boss。看着好欺负？打到半血它"假死"复活变狂暴，别得意太早。掉落：赤龙首、焰鳞甲、龙息
- **焚天龙皇**（800 血）— BossRush 的最终 Boss。7 种技能、双阶段、1 血还能喊儿子来护驾。掉落一堆毕业装：龙王之冕、龙王鳞铠、腾云驾雾图腾、逆鳞、焚皇断界戟、焚天龙铳
- **幽灵女巫**（1000 血）— 最阴险的 Boss。隐身闪现、镰刀劈脸、诅咒领域、还能召唤亡灵小弟。掉落：噬魂挽歌

# 在哪能遇到它们？

- **标准 BossRush / 无间炼狱** — 前 20 波不会出现三大原创 Boss（给你热手），之后正常参与抽取
- **白手起家** — 前 5 波没 Boss，6-15 波屏蔽强力 Boss，16 波后全 Boss 池开放
- **划地为营** — 龙裔遗族每局最多 1 只，焚天龙皇不参与，幽灵女巫正常抽取
- **血猎追击** — 同划地为营规则，另外 Boss 死了会自动补充新的

# Boss 太烦？筛掉它

按 **Ctrl+F10** 打开 Boss 筛选器：

- 直接禁用某个 Boss，让它永远别出现
- 调整各 Boss 在无间炼狱中的出现权重

详见 Boss 筛选器。
