# 焚天龙皇

# 这位爷是谁？

**BossRush 里的最终 Boss。** 悬浮在半空中，7 种技能轮着招呼你，打到 1 血还能喊儿子来护驾——这就是焚天龙皇，一条让无数玩家怀疑人生的空中霸主。

但它的掉落表也是最豪华的：龙王套装、飞行图腾、断界戟……**你的毕业装，全在它身上。**

- **血量**：800
- **伤害倍率**：0.3x
- **碰撞伤害**：15（范围 1.5 米）
- **悬浮高度**：3-5 米（对，它飘在你头顶上打你）
- **跟随速度**：8 单位/秒

# 它会什么技能？（一共 7 个）

# 棱彩弹
向你射出 8 颗追踪弹，追踪 2.5 秒后变直线。每颗 15 点伤害。横向移动可以甩开。

# 螺旋棱彩弹（二阶段限定）
每 0.1 秒发一颗，持续 1 秒，角度递增 15°——弹幕游戏既视感。

# 冲锋
0.8 秒蓄力后高速俯冲，蓄力末尾地面出现预警圆环。冲锋伤害 30 点，路径留下熔浆轨迹（每 0.5 秒 5 点，持续 3 秒）。**二阶段连冲两次**，躲完一次别停。

# 日轮舞
瞬移到你身边，释放 6 道旋转光束（30°/秒），分 3 波偏移 20°，配合 24 方向旋转弹幕。看着很花哨，找到光束间隙站好就行。

# 万彩虹环
13 颗星体螺旋扩散到 10 米后收缩，轨迹 5 点伤害，旋转 45°/秒，持续 8 秒。**扩散时往外跑，收缩时往里靠**。

# 虚影枪
12 根长枪先出预警线（1 秒），然后以 40 速射出，每根 25 点。预警线就是安全提示——**站在线和线之间**。

# 虚影枪阵（二阶段限定）
升级版——4 波 ×16 根长枪，波间隔 0.5 秒，覆盖范围更大。二阶段的你要习惯满屏都是枪。

# 两个阶段

# 第一阶段（满血 → 50%）

- 攻击间隔 1.0 秒，还算从容
- 技能循环：棱彩弹 → 冲锋 → 日轮舞 → 冲锋 → 万彩虹环 → 棱彩弹 → 冲锋 → 虚影枪 → 冲锋 → 万彩虹环 → 循环

# 第二阶段（50% 以下）

- 攻击间隔缩到 0.5 秒——**频率翻倍**，你的反应时间直接砍半
- 解锁螺旋棱彩弹和虚影枪阵
- 技能循环：虚影枪阵 → 棱彩弹 → 冲锋 → 万彩虹环 → 棱彩弹 → 日轮舞 → 虚影枪 → 冲锋 → 螺旋棱彩弹 → 循环
- 阶段转换时龙皇会瞬移到你头顶，显示狂暴提示——**准备好**

# "孩儿护我！"——最骚的保命机制

打到 1 血，龙皇不会死。它会飞升到 5 米高空喊出：**"孩儿护我！"**

然后一只属性减半的龙裔遗族从天而降，回应：**"爹爹！"**

- 龙裔遗族活着的时候，龙皇每 3 秒发射棱彩弹
- 你必须先杀龙裔遗族，龙皇才会真正倒下

::: warning
别想着无视龙裔遗族直接对龙皇输出——它 1 血锁定，你打不死的。先收拾儿子。
:::

# 掉落——BossRush 的毕业装全在这

- **腾云驾雾图腾** — 15% 掉率
- **龙王之冕**（头盔）— 15% 掉率
- **龙王鳞铠**（护甲）— 15% 掉率
- **逆鳞**（图腾）— 35% 掉率（最容易出的）
- **焚皇断界戟**（近战）— 15% 掉率
- **焚天龙铳**（枪械）— 5% 掉率（极其稀有，拿到请截图留念）

# 怎么打？

- **用枪**。它飘在空中 3-5 米，近战基本够不到
- **冲锋**有预警圆环，看到就横向移动，别站桩
- **日轮舞**的光束有间隙，找缝站。如果找不到就往外跑出范围
- **万彩虹环**先扩后缩——扩散阶段跑出去，收缩阶段跑回来
- **二阶段**攻击频率翻倍，保持移动是唯一正解
- **"孩儿护我"阶段**先杀小的，注意躲龙皇的棱彩弹
- **穿龙裔/龙王套装**可以免疫火焰伤害 + 回血，难度骤降
- 想练无伤？用 Boss 筛选器清空其他 Boss，一对一单挑

# 在哪能打？

- **标准 BossRush / 无间炼狱** — 前 20 波不出现，之后正常参与
- **划地为营** — 不会出现（太猛了）
- **血猎追击** — 不会出现
